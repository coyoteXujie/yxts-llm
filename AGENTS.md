# AGENTS.md

This file provides guidance to Codex (Codex.ai/code) when working with code in this repository.

## Project Overview

这是 **白金英雄坛说 (YXTS-LLM)** 的现代 Python 重制版，基于 Arcade 3.0.2 引擎开发。
原版是一款基于 6502 汇编的武侠 MUD 游戏，运行在文曲星掌上设备上。

- **原版源码**: `based-src/` 目录
- **Python参考实现**: `platinum_heroes_merged/` 目录
- **当前项目**: `src/` 目录

## Common Commands

- `python main.py`: 启动游戏
- `pip install -r requirements.txt`: 安装依赖

## Architecture

- `src/core/entities.py`: 实体定义 (Player, NPC, Item, Skill, Faction, NpcType 等)
- `src/core/world.py`: 游戏世界 (地图、NPC、物品、武功、绝技、对话)
- `src/core/renderer.py`: 渲染器 (瓦片绘制、NPC形象、建筑标签)
- `src/core/window.py`: 主窗口 (角色创建、移动、交互、UI)
- `src/core/combat.py`: 战斗系统
- `src/core/quest.py`: 任务系统
- `src/core/event.py`: 事件系统
- `src/core/config.py`: 配置

## Key Technical Notes

- **Arcade 3.0.2 API**: 使用 `on_update()` 而非 `update()`，使用 `draw_rect_filled()` / `LBWH()` 等
- **坐标系统**: 世界坐标(像素) ↔ 瓦片坐标(网格) ↔ 屏幕坐标，Y轴翻转
- **瓦片类型**: 50种 (0-49)，WALKABLE集合定义可通行瓦片
- **Faction枚举**: NONE/BAGUA/FLOWER/HONGLIAN/NAJA/TAIJI/XUESHAN/XIAOYAO
