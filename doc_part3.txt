### 2.4 那迦派

| 项目 | 内容 |
|------|------|
| **位置** | 东北·幽州 |
| **掌门** | 钟央（40岁） |
| **门派特色** | 以忍术、暗杀和情报收集见长，行事诡秘 |
| **碎片保管** | 碎片藏于幽州地下密室，由机关守护 |
| **门派声望** | ★★☆☆☆（令人畏惧） |
| **势力规模** | 门徒约300人（精锐） |

**门派简介**：

那迦派是七派中最神秘的存在。门派位于东北幽州深山之中，终年浓雾弥漫，外人难以寻觅。那迦派源自东瀛忍术与中原武学的融合，门中弟子擅长隐匿、暗杀、情报刺探，是江湖中最令人忌惮的势力。那迦派从不参与江湖纷争，但一旦出手，必是致命一击。

**掌门钟央**：人称"千面影"，据说精通易容术，能以假乱真。极少有人见过他的真面目，甚至门中弟子也只闻其声不见其人。钟央与暗影司之间有着复杂的关系——那迦派的情报网络与暗影司的谍报系统时有交集，双方既有合作也有对抗。钟央似乎在下一盘很大的棋，但无人知晓他的真正目的。

**核心武学**：
- 影遁术：隐匿身形的绝技，可近乎完全消失于环境之中
- 千影刺：暗杀绝技，一击必杀
- 幻心术：精神控制之术，可短暂操控他人心智

**内部矛盾**：那迦派内部实行严格的等级制度，"影首"（掌门）拥有绝对权威。然而，二把手**夜叉**对钟央的行事方式越来越不满，认为他过于保守，错失了扩张势力的良机。夜叉暗中与暗影司达成交易，出卖门中情报以换取权力。

### 2.5 太极门

| 项目 | 内容 |
|------|------|
| **位置** | 北方·恒州 |
| **掌门** | 清虚道人（75岁） |
| **门派特色** | 历史最悠久的门派，以太极功法为核心，讲究以柔克刚 |
| **碎片保管** | 碎片封于太极门镇派之宝"太虚镜"背面 |
| **门派声望** | ★★★★★ |
| **势力规模** | 门徒约1200人 |

**门派简介**：

太极门是七派中历史最悠久、声望最高的门派，坐落于北方恒州武当山上。门派传承可追溯至三百年前，历代掌门皆为当世一流高手。太极门以"道"为根本，武学讲究阴阳调和、以柔克刚，看似平淡无奇，实则蕴含天地至理。二十年前，正是清虚道人倡议七派结盟，才有了赤霞之战的胜利。

**掌门清虚道人**：七派掌门中年纪最大、辈分最高者，被尊为"武林泰斗"。表面上仙风道骨、超然物外，实则城府极深。清虚道人是赤霞之盟的发起者，也是碎片保管制度的设计者——但没有人知道，他之所以如此安排，是因为他早已在暗中研究碎片的力量，试图将七块碎片的力量融为一体，创造超越屠龙刀的全新神兵。清虚道人与暗影司司主沈夜之间，存在着一段跨越三十年的师徒关系——沈夜，正是清虚道人早年收养的孤儿。

**核心武学**：
- 太极拳：以柔克刚，借力打力
- 太极剑：剑法圆融，攻守一体
- 太虚神功：太极门至高内功，可吸纳他人内力为己用

**内部矛盾**：清虚道人的秘密研究已被部分核心弟子察觉。大弟子**玄真**对师父的忠诚开始动摇，而二弟子**玄清**则被清虚道人秘密培养为碎片融合实验的"容器"。太极门表面平静，实则暗藏惊天秘密。

### 2.6 雪山派

| 项目 | 内容 |
|------|------|
| **位置** | 西北·昆仑雪山 |
| **掌门** | 白瑞德（50岁） |
| **门派特色** | 以冰雪系武学为主，门人性格冷傲孤僻 |
| **碎片保管** | 碎片被封入万年玄冰之中，藏于雪山之巅 |
| **门派声望** | ★★★★☆ |
| **势力规模** | 门徒约600人 |

**门派简介**：

雪山派位于西北昆仑雪山之巅，海拔极高，常年冰雪覆盖，寻常人等根本无法到达。雪山派门人长期在极端环境中修炼，内力深厚且属性偏寒，出手时寒气逼人，可冻伤对手经脉。雪山派与外界极少往来，行事孤傲，在七派中最为封闭。

**掌门白瑞德**：人称"冰魄剑"，性格冷峻寡言，不喜交际。白瑞德是七派掌门中武艺最高者之一，其"冰魄寒光剑"一出，方圆十丈内寒气凝结，对手未战先衰。白瑞德对赤霞之盟的态度最为冷淡——他不在乎碎片的去向，只在乎雪山派不被外界打扰。然而，暗影司的触手已经伸到了昆仑雪山，白瑞德的独子**白少寒**在一年前离奇失踪，白瑞德怀疑是暗影司所为，但苦无证据。

**核心武学**：
- 冰魄寒光剑：剑法凌厉，附带冰冻效果
- 雪崩掌：掌力雄浑，如雪崩般不可阻挡
- 玄冰诀：内功心法，可将内力转化为极寒之气

**内部矛盾**：白少寒的失踪在门中引起恐慌，部分弟子认为应当下山追查，白瑞德却坚持按兵不动。副掌门**冷千秋**主张主动出击，与白瑞德的保守策略产生冲突。冷千秋的真实身份，是暗影司安插在雪山派的卧底。
