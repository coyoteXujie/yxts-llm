### 2.7 逍遥派

| 项目 | 内容 |
|------|------|
| **位置** | 行踪不定（据点遍布天下） |
| **掌门** | 逍遥子（年龄不详） |
| **门派特色** | 秘密门派，以情报和谋略对抗暗影司，门人隐藏于各行各业 |
| **碎片保管** | 无碎片（逍遥派未参与赤霞之战） |
| **门派声望** | ★☆☆☆☆（几乎无人知晓其存在） |
| **势力规模** | 核心成员约50人，外围联络人约200人 |

**门派简介**：

逍遥派是七派之外的隐藏势力，未参与二十年前的赤霞之战，因此没有分得碎片。逍遥派的创始人是逍遥子的师兄——**无涯子**。无涯子与逍遥子本是同门师兄弟，二十年前，无涯子预见到屠龙刀碎片必将引来新的纷争，主张将碎片彻底销毁。但七派掌门贪恋碎片之力，拒绝了无涯子的建议。无涯子含恨离去，创立逍遥派，发誓要阻止碎片重铸。无涯子已于十年前病逝，逍遥子继承遗志，继续与暗影司周旋。

**掌门逍遥子**：身份成谜，无人知晓其真实姓名和来历。逍遥子精通易容、变声，能以不同身份出现在不同场合。他（或她）似乎对所有人的秘密都了如指掌，却从不暴露自己的底牌。逍遥子与暗影司司主沈夜之间有着极深的渊源——两人曾是至交好友，后因理念不同而分道扬镳。

**核心武学**：
- 逍遥游：身法绝学，行动如风，来去无踪
- 北冥神功：可吸纳他人内力，化为己用
- 凌波微步：步法精妙，可躲避一切攻击

**内部矛盾**：逍遥派内部对"如何阻止碎片重铸"存在分歧。一派主张以情报和谋略暗中破坏暗影司的计划，另一派则主张直接出手抢夺碎片、先行销毁。逍遥子倾向于前者，但激进派代表**风无痕**越来越难以约束。

### 2.8 暗影司（反派势力）

| 项目 | 内容 |
|------|------|
| **位置** | 京城·暗影司总部（地下） |
| **司主** | 沈夜（45岁） |
| **组织特色** | 朝廷秘密谍报机构，擅长渗透、离间、暗杀 |
| **目标** | 收集七块碎片，重铸屠龙刀，控制武林 |
| **势力规模** | 核心成员约100人，外围眼线遍布天下 |

**组织简介**：

暗影司是大梁皇帝亲自下旨成立的秘密机构，名义上负责"稽查江湖事务"，实则是朝廷控制武林的暗棋。暗影司司主沈夜，原是清虚道人收养的孤儿，自幼在太极门中长大，后因触犯门规被逐出师门。沈夜流落京城，被朝廷重臣看中，从此踏入仕途。他凭借过人的智谋和手段，一步步爬上暗影司司主之位。

沈夜的真正目的并非仅仅为朝廷效力——他想要重铸屠龙刀，不是为了皇帝，而是为了自己。二十年前，他的亲生父母正是死于血月教之手，他深信只有屠龙刀的力量才能彻底铲除像血月教这样的邪恶势力。为此，他不惜一切代价。

**核心人物**：
- **沈夜**（司主）：智谋深远，手段狠辣，但内心深处仍有一丝善意
- **墨鸦**（副司主）：沈夜的左右手，冷酷无情的执行者
- **绣花女**（情报头子）：以绣娘身份潜伏于各派之中，收集情报
