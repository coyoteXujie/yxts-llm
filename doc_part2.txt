## 二、势力设定

### 2.1 八卦门

| 项目 | 内容 |
|------|------|
| **位置** | 西北·乾州 |
| **掌门** | 韦扬（58岁） |
| **门派特色** | 以八卦掌、八卦刀为核心武学，讲究步法与方位的配合 |
| **碎片保管** | 碎片嵌于八卦刀祖师堂供奉的"定乾刀"刀柄之中 |
| **门派声望** | ★★★★☆ |
| **势力规模** | 门徒约800人 |

**门派简介**：

八卦门坐落于西北乾州天罡山上，山势险峻，云雾缭绕。门派建筑依山而建，呈八卦方位排列，外人入山若不懂八卦步法，极易迷失其中。八卦门以步法见长，"走转"之术独步天下，攻守之间如行云流水，令对手难以捉摸。

**掌门韦扬**：人称"乾坤手"，性格刚正不阿，在七派掌门中资历最深。年轻时曾参与赤霞之战，亲手从厉无极身上取下碎片。近年来年事渐高，门中事务多交由大弟子**韩铁衣**打理。韦扬坚守赤霞之盟，拒绝任何重铸碎片的提议，但韩铁衣却暗中与暗影司有所往来……

**核心武学**：
- 八卦游身掌：以走转为根基，掌法连绵不绝
- 乾坤八卦刀：刀法刚猛，配合步法可攻八方
- 天罡步：身法绝技，移动时如鬼魅般难以捕捉

**内部矛盾**：大弟子韩铁衣主张利用碎片的力量壮大八卦门，与掌门韦扬的保守立场形成对立。暗影司正是利用这一矛盾，在门中培植势力。

### 2.2 花间派

| 项目 | 内容 |
|------|------|
| **位置** | 南方·越州 |
| **掌门** | 清照（已故）→ 隐娘（现任，32岁） |
| **门派特色** | 以医术、毒术和暗器闻名，亦正亦邪 |
| **碎片保管** | 碎片被封入"百花丹"的药引之中，藏于花间秘库 |
| **门派声望** | ★★★☆☆ |
| **势力规模** | 门徒约500人 |

**门派简介**：

花间派位于南方越州百花谷中，四季如春，百花盛开。门中弟子多为女性，以医术行走江湖，救死扶伤，深受百姓爱戴。然而，花间派的另一面却鲜为人知——她们的毒术同样出神入化，暗器"花针"更是令人闻风丧胆。江湖中有言："花间救人一命，花间夺命无声。"

**前任掌门清照**：人称"百花仙子"，医术通神，曾救活无数江湖人士。三年前突然暴毙，死因成谜。官方说法是旧伤复发，但门中弟子私下议论纷纷，有人怀疑是暗影司下毒所致。

**现任掌门隐娘**：清照的关门弟子，年仅32岁便接掌花间派。性格冷静沉稳，看似柔弱实则心机深沉。她一直在暗中调查师父的死因，同时竭力维持门派在七派中的地位。隐娘手中掌握着一份清照临终前留下的密信，内容关乎碎片的秘密……

**核心武学**：
- 百花掌：掌法轻柔，暗含毒劲
- 花针术：暗器绝技，细如牛毛，淬以剧毒
- 回春诀：花间派独门疗伤内功，可快速恢复伤势

**内部矛盾**：部分长老认为隐娘年纪太轻、资历太浅，不服其掌门之位。大长老**柳如烟**表面恭顺，实则暗中拉拢门中势力，意图取而代之。

### 2.3 红莲教

| 项目 | 内容 |
|------|------|
| **位置** | 东南·闽州 |
| **教主** | 于红儒（45岁） |
| **门派特色** | 草根门派，以民间疾苦为己任，讲究义气与热血 |
| **碎片保管** | 碎片被铸入红莲教圣物"火莲灯"底座之中 |
| **门派声望** | ★★★☆☆（民间声望极高） |
| **势力规模** | 教众约3000人（含大量非战斗人员） |

**门派简介**：

红莲教不同于其他六派，它并非传统意义上的武林门派，而是一个扎根民间的草根组织。教众多为穷苦百姓、流民、手工业者，他们因不堪官府压迫和豪强欺凌而聚集在一起，以"红莲"为号，行侠仗义、劫富济贫。红莲教在东南沿海一带声望极高，百姓视其为守护神，官府却将其视为眼中钉。

**教主于红儒**：出身贫寒，少年时全家被贪官害死，流落江湖，后被红莲教前教主收为弟子。他性格豪爽直率，嫉恶如仇，在民间有"红莲大侠"之称。于红儒对赤霞之盟的态度最为激进——他认为碎片应当被销毁，因为任何力量一旦被垄断，最终都会成为压迫的工具。

**核心武学**：
- 红莲掌法：刚猛霸道，以力破巧
- 烈火刀：大开大合，气势如虹
- 义民心法：独特的内功心法，越战越勇，伤势越重内力越强

**内部矛盾**：红莲教内部存在"温和派"与"激进派"之争。温和派主张与朝廷妥协以保全教众，激进派则主张武装反抗。暗影司利用这一分歧，在教中煽动激进派制造事端，为朝廷出兵镇压制造借口。
