## 🌟 PixelSRPG-Forge - Free Pixel Art Assets for SRPG Games

A comprehensive, continuously updated collection of high-quality pixel art assets specifically designed for Strategy Role-Playing Games (SRPGs). Everything you need to build your dream tactical game!

### 🎯 What's Included?

| Category | Content | Status |
|----------|---------|---------|
| 🏰 **Buildings** | Castles, towns, dungeons, shops, etc. | ✅ Available |
| 🐾 **Creatures & Animals** | Fantasy monsters, pets, wildlife | ✅ Available |
| 👥 **Characters** | Heroes, villains, NPCs with animations | ✅ Available |
| ⚔️ **Weapons & Equipment** | Swords, armor, magical items | ✅ Available |
| 🎭 **Costumes & Apparel** | Character customization options | ✅ Available |
| ✨ **Skill Effects** | Magic spells, combat animations | ✅ Available |
| 🎯 **Skill Icons** | Ability icons, status effect icons | ✅ Available |
| 🎵 **Sound Effects** | UI sounds, combat sounds, ambiance | ✅ Available |
| 🗺️ **Terrain Tiles** | Grass, water, mountains, forests, etc. | ✅ Available |
| 📊 **UI Elements** | Buttons, frames, health bars, menus | ✅ Available |

### 🚀 Quick Start

1. **Download the entire repository**
   ```bash
   git clone https://github.com/Huu-Yuu/PixelSRPG-Forge.git
   ```

2. **Or download specific categories**
- Browse folders and download only what you need
- All assets are organized by category

```
PixelSRPG-Forge/

├── Skill_Effects_技能特效/  # Magic effects and combat animations
├── Skill_Icons_技能图标/    # Ability and status effect icons
├── Sound_Effects_音效音乐/  # UI, combat and ambient sounds
├── Terrain_地形地块/        # Terrain tiles (grass, water, mountains, etc.)
├── UI_Elements_界面元素/    # Buttons, frames, health bars, menus

```

### 🛠️ Tools

**PNG Icon Packer**
A utility tool to merge multiple PNG icons into a single sprite sheet. Features include:
- Customizable icon dimensions
- Automatic or manual grid arrangement
- Transparency preservation
- Index file generation for easy reference
- Perfect for game development texture atlasing

See [png_icon_packer_usage.md](png_icon_packer_usage.md) for detailed instructions.
