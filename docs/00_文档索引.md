## 文档约定

1. 设计文档记录完整目标；`14_Godot实现差距审计.md` 记录当前落地程度。
2. 技术文档以 `godot_project/` 为准，不再引用已清理的旧 Python/Pygame 运行链路。
3. 新增或修改 NPC、区域、物品、任务、美术映射后，需要同步更新对应设计文档或审计文档。
4. 每次大改后优先运行 `tools/validate_godot_data.py` 和 Godot 头less启动验证。
