#### LOD（细节层次）

```python
def get_lod_distance(self, particle_x, particle_y, camera_x, camera_y):

    distance = sqrt((particle_x - camera_x)**2 + (particle_y - camera_y)**2)
```

#### 对象池

```python
class ParticlePool:
    """粒子对象池"""
    
    def __init__(self, size: int = 500):
        self.pool = [Particle() for _ in range(size)]
        self.available = list(range(size))
        self.in_use = set()
    
    def acquire(self) -> Optional[Particle]:
        """获取一个可用粒子"""
        if self.available:
            idx = self.available.pop()
            self.in_use.add(idx)
            return self.pool[idx]
        return None
    
    def release(self, particle: Particle):
        """释放粒子回池"""
        idx = self.pool.index(particle)
        if idx in self.in_use:
            self.in_use.remove(idx)
            self.available.append(idx)
```

**对象池优势：**
- 避免频繁创建/销毁对象
- 减少GC压力
- 提高性能稳定性

#### 距离雾

```python
class DistanceFog:
    """距离雾效果"""
    
    def apply_fog(self, pixel_x, pixel_y, camera_x, camera_y, 
                  fog_color, fog_density):
        """
        应用距离雾
        
        公式：
        fog_factor = 1 - exp(-distance * fog_density)
        final_color = pixel_color * (1 - fog_factor) + fog_color * fog_factor
        """
        distance = sqrt((pixel_x - camera_x)**2 + (pixel_y - camera_y)**2)
        fog_factor = 1 - math.exp(-distance * fog_density / 1000)
        
        return self._blend_color(pixel_color, fog_color, fog_factor)
```

**距离雾特点：**
- 随距离增加，雾效增强
- 指数增长，远处完全淹没
- 增强场景深度感

#### 高度雾

```python
class HeightFog:
    """高度雾效果（用于山谷、低地）"""
    
    def apply_height_fog(self, pixel_y, ground_y, fog_color, fog_density):
        """
        应用高度雾
        
        公式：
        height_factor = max(0, (ground_y - pixel_y) / fog_height)
        """
        if pixel_y > ground_y:
            return pixel_color  # 高于地面，无雾
        
        height_factor = (ground_y - pixel_y) / 200  # 200像素为雾高度
        height_factor = min(1.0, height_factor * fog_density)
        
        return self._blend_color(pixel_color, fog_color, height_factor)
```

**高度雾特点：**
- 低处雾浓，高处雾淡
- 适合山谷、盆地场景
- 营造神秘氛围

#### 体积雾

```python
class VolumetricFog:
    """体积雾效果"""
    
    def apply_volumetric_fog(self, ray_start, ray_end, fog_density):
        """
        应用体积雾（光线行进）
        
        公式：
        transmittance = exp(-optical_depth)
        optical_depth = integral(fog_density * step_size)
        """
        steps = 10
        step_size = distance(ray_start, ray_end) / steps
        optical_depth = 0
        
        for i in range(steps):
            pos = lerp(ray_start, ray_end, i / steps)
            density = self._sample_fog_density(pos)
            optical_depth += density * step_size
        
        transmittance = math.exp(-optical_depth)
        return transmittance
```

**体积雾特点：**
- 模拟光线在雾中传播
- 光束效果明显
- 性能开销较大

### 8.2 3D音效定位

```python
class SpatialAudio:
    """3D空间音效系统"""
    
    def play_3d_sound(self, sound, source_x, source_y, player_x, player_y):
        """
        播放3D定位音效
        
        参数：
        - sound: 音效资源
        - source_x, source_y: 声源位置
        - player_x, player_y: 玩家位置
        """
        # 计算距离和方向
        dx = source_x - player_x
        dy = source_y - player_y
        distance = sqrt(dx**2 + dy**2)
        angle = atan2(dy, dx)
        
        # 距离衰减
        volume = self._calculate_attenuation(distance)
        
        # 声相定位
        pan = cos(angle)  # -1（左）到 1（右）
        
        # 播放
        sound.play(volume=volume, pan=pan)
    
    def _calculate_attenuation(self, distance):
        """计算距离衰减"""
        max_distance = 1000  # 最大听觉距离
        if distance >= max_distance:
            return 0
        return 1 - (distance / max_distance)**2
```

**3D音效应用：**
- 雷声：根据闪电位置定位
- NPC对话：根据NPC位置定位
- 战斗音效：根据敌我位置定位

#### 音乐过渡效果

```python
def crossfade(self, from_track, to_track, duration=2.0):
    """
    交叉淡入淡出
    
    参数：
    - from_track: 当前音轨
    - to_track: 目标音轨
    - duration: 过渡时长（秒）
    """
    steps = int(duration * 60)  # 60fps
    
    for i in range(steps):
        t = i / steps
        from_track.volume = 1 - t
        to_track.volume = t
        yield  # 等待下一帧
    
    from_track.stop()
    to_track.volume = 1
```
