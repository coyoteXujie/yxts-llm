# 白金英雄坛说 - Godot 实现差距审计

> 审计日期：2026-05-31
> 审计范围：以 `docs/00_文档索引.md` 中的新版主文档为准，对照 `godot_project/` 当前 Godot 4 工程。旧版文档和旧原型目录已清理，不作为当前必须实现项。

## 1. 当前实现结论

Godot 版目前是一个可玩的 Alpha 竖切：已经有主菜单、角色创建、连续世界地图、73 个结构化区域、区域探索度、世界地图到局部区域的进入/返回、局部区域相邻转场、局部区域到商铺内景的进入/返回、可互动地标/资源点、驿亭休整与危险野外伏击事件、发现面板、带 NPC/任务标记和点击选择的世界地图面板、基础目的地标记、带路线/耗时/费用/风险提示和基础风险结算的城镇/门派快速旅行、玩家移动、99 个 NPC、对话、教学、商店、背包、带命中/暴击/冷却/状态/敌人风格/掉落/首领阶段的基础战斗、33 个带前置门槛和后续提示的任务、七派各 3 段门派入门任务线、简化修炼、存档读档、江湖传闻/世界事件记录、半俯视 2.5D 阴影/道具投影/景深氛围层、对话世界导演与可选 LLM 后端，以及一批可运行的生成美术资源。

但它距离设计文档中的完整武侠世界还有较大差距。当前主要完成的是“能跑起来的主循环”，文档中的区域系统、剧情系统、复杂战斗、经济声望、NPC 关系、天气氛围、完整 UI 和大规模美术资产，大多还没有在 Godot 中落地。

## 4. 主要未实现项

### 4.1 世界观与势力

- 七大门派已有名称、基础技能表、部分 NPC 分布和各 3 段入门任务线；门派历史、组织结构、门规、声望奖励、内部矛盾没有系统化实现。
- 中立势力没有实现：天机阁、铁血盟、药王谷、暗影司、血月教、龙门客栈。
- 五大城池的文化、经济差异、特色 NPC、隐藏区域、动态事件没有对应系统。
- 已有轻量江湖传闻/世界事件记录；还没有季节事件、随机世界事件、势力冲突事件和玩家选择造成的复杂世界变化。

### 4.2 世界地图

- 当前世界层仍是压缩连续大地图，不是文档要求的 73 个完整独立区域场景。
- 已有区域 ID 数据结构、进入识别、局部区域模板、相邻区域路牌转场、可互动地标/资源点和独立商铺内景；局部地图已按地貌和区域类型做第一轮差异化，但还没有为 73 个区域逐一定制的正式地图布局、隐藏点和事件层。
- 已有基础探索度记录、一次性地标奖励、每日刷新资源点和发现面板；还没有隐藏区域发现、区域称号、区域 Boss 解锁。
- 已有发现区域后的城镇/门派基础快速旅行、路线耗时/风险/费用估算、旅费扣除、旅行时间推进和高风险路线基础后果；还没有传送阵、通行证限制和完整野外旅行事件链。
- 已有大地图面板、HUD 小地图、NPC/任务目标标记、目的地标记和基础路线提示；还没有筛选和完整路线规划。
- 五大城池、主要城镇、七大门派已有第一版局部地图、相邻区域入口和商铺/功能入口模板；还没有按城市文化差异做完整街区、府衙、武馆、黑市、门派内部建筑和专属事件。

### 4.3 NPC 系统

- `03_npc_design.md` 中 31 个明确命名的核心/门派 NPC 已落地 10 个：苏梦瑶、陈天行、赵无极、玄机子、花如玉、烈火、蛇王、太极真人、冰魄、逍遥子；其余核心 NPC 仍未实现。
- 已有 NPC 性格字段展示、对话关系侧栏、基础好感度、交互记忆保存、地图环境气泡、跨天传闻记忆扩散和基于世界状态/任务/关系/江湖传闻的动态闲谈；还没有完整 8 维性格模型、人物弧光、日常作息和关系网络传播。
- 已有短期对话记忆、关系语气变化和任务/发现事件形成的江湖传闻；还没有救助、背叛、亲友死亡、声望变化后的复杂长期世界反馈。
- 城镇 NPC 模板已有少量抽象外观，但没有模板化作息、服务类型、可复用任务池。
- 敌人只有静态地图敌人，没有文档中的山贼、流寇、邪教徒、野兽、精怪、朝廷鹰犬体系。

### 4.4 物品与装备

- 当前 Godot 只有 22 个物品，文档目标是 400+ 物品。
- 物品分类不完整：没有材料、秘籍、任务物品、宝物、特殊货币、毒药体系。
- 没有稀有度、鉴定、耐久、修理、强化、镶嵌、套装、打造、炼丹、采集制作。
- 装备槽只有武器和防具，文档要求头部、身体、手部、脚部、饰品等多槽位。
- 没有门派契合度加成和传说武器独特技能。
- 没有出售物品、交易、掉落表、隐藏宝箱。

### 4.5 战斗

- 当前战斗仍是 1v1 回合制，已有基础命中、暴击、冷却、状态、敌人风格、掉落和首领半血阶段，但还没有完整八阶段状态机。
- 文档中的 30+ 武学招式只保留了技能名，未实现招式效果、MP 消耗、冷却、伤害倍率、状态附加。
- 暴击、命中、闪避和少量连击已接入；还没有五行/门派属性克制、正式格挡、反击和完整连招体系。
- 已有流血、中毒、灼烧、眩晕、迟缓、虚弱、破绽、守势等基础状态；还没有冰冻、内伤、状态抗性和状态图标完整表现。
- 已有 brawler/brute/bandit_boss/assassin/beast/boss 风格 AI、回血、守势和半血阶段；还没有完整决策树、门派 AI、协作战术和正式多阶段 Boss。
- 已有危险野外驿亭伏击、区域遇敌选择和敌人掉落；还没有世界 Boss、持续刷怪、等级缩放和完整掉落表。
- 没有阵型、前后排、地形战斗加成。
- 战斗 UI 已有敌人头像、武学图标按钮、战斗日志和轻量浮字反馈；仍没有正式立绘、状态图标、技能子菜单、战斗动画和水墨特效。

### 4.6 修炼

- 当前只是 `learn_skill` 和 `practice_skill`，没有独立修炼系统类。
- 没有八大内功的属性曲线、修炼条件、特殊效果。
- 没有五大轻功、移动速度/闪避加成公式。
- 没有经脉系统、八脉打通、突破路线、突破消耗。
- 没有大境界突破、天象变化、悟道事件、走火入魔风险。
- 没有师徒传承、秘籍传承、顿悟传承。
- 没有修炼与装备、战斗属性的完整联动。
- 技能面板不是技能树，只是列表升级。

### 4.7 经济与声望

- 当前只有单一银两，没有文/两/金三种货币单位和兑换关系。
- 商店已有客栈、药铺、铁匠铺、布庄、市集、茶肆等第一批类型入口和内景，但商品、折扣、出售、修理、黑市、地区差价仍很浅。
- 没有动态市场、供需、季节价格、事件价格、地区差价套利。
- 没有拍卖行、黑市、钱庄、存款利息、贷款、钱庄信誉。
- 没有门派声望、城市声望、NPC 好感、名望等级。
- 没有正邪值影响系统；当前只有 `daode` 数值奖励，缺少实际规则影响。

### 4.8 剧情与奇遇

- 当前已有 33 个 Alpha 任务和一条可连续推进的主线开端，并已加入前置门槛、等级门槛、门派限制、新手自动追踪、后续提示、任务完成传闻和武学奖励；但还不是文档中的五章主线。
- 没有八阶段剧情架构、主线节点图、剧情选择、分支后果。
- 没有八种结局和 New Game+。
- 没有 50+ 支线任务、任务失败后果、任务时间限制。
- 没有八大随机奇遇事件和权重触发机制。

## 6. 近期最合理的下一步

建议下一轮不要直接继续堆 NPC 或美术，而是把已接入的“世界地图 -> 局部区域 -> 商铺内景”扩展成数据驱动区域系统。原因是地图、NPC、任务、奇遇、快速旅行、天气、资源点都依赖区域 ID；没有这个底座，后续内容会继续散落在坐标和硬编码里。

1. 已完成 `data/regions.json`，把 5 城、16 镇、45 野外、7 门派全部登记为结构化区域。
