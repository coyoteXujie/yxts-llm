## 二、各系统图片需求详细描述

### 2.1 区域场景背景（73 个区域）

**当前状态**：每个区域有一个横版舞台背景 PNG

**需求**：每个区域需要 4 层 PNG：
- 后景（远景山脉/天空）
- 中景（城镇/建筑轮廓）
- 地面（街道/田野）
- 前景（屋檐/树木前景遮挡）

**详细描述模板**：

```markdown
## [区域名称]

### 后景层（1920x1080）
- 天空颜色：[描述]
- 远景元素：[山脉/云朵/日出/夕阳等]
- 色调：[暖色/冷色/水墨灰]

### 中景层（同上）
- 建筑轮廓：[中式建筑/商铺/门派宫殿]
- 特色元素：[旗帜/灯笼/招牌]
- 高度感：[多层楼宇/单层瓦房]

### 地面层（同上）
- 地面材质：[石板/泥土/青砖]
- 道路细节：[主街/小巷/河道]
- 装饰物：[摊位/水井/树木]

### 前景层（同上）
- 遮挡元素：[屋檐角/树枝/灯笼]
- 透视关系：[近大远小]
- 透明度：[30-50% 半透明]
```

**示例 - 清河镇**：

```markdown
## 清河镇

### 后景层（1920x1080）
- 天空：淡青色晨雾天空，西方略泛鱼肚白
- 远景：远处青山轮廓如黛，近处丘陵起伏
- 色调：水墨淡彩，晨雾朦胧

### 中景层
- 建筑：密集木构小镇，灰瓦白墙
- 特色：茶肆"醉仙楼"招牌、酒旗飘扬
- 高度：2-3层木质阁楼，鳞次栉比
- 细节：灯笼高挂、窗格雕花、木质招牌

### 地面层
- 地面：青石板主街，中央略凹
- 道路：东西向主街，南北小巷
- 装饰：街边水井、木质栅栏、货物堆

### 前景层
- 遮挡：近景屋檐角探入画面左上和右上部
- 元素：下垂的杨树枝条、门帘边角
- 透明度：40% 半透明，不遮挡主体
```
