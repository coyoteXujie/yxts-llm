## 一、世界地图总览

### 1.1 大陆地理概述

白金英雄坛说的世界以中原大地为核心，五大城市如同五颗明珠镶嵌于大陆之上，由官道、水路、山径相连。秦岭分割南北，太行山纵贯中原东侧，武当山耸立荆楚，青城峨眉雄踞巴蜀。黄河自西向东奔流，长江横贯南方，渭水蜿蜒于关中平原，岷江滋润天府之国。

世界分为五大城市体系，每个城市辐射若干卫星镇，每个镇连接2-3个野外区域，门派则隐藏在野外深处。层级关系：城市数量 << 小镇数量 < 野外数量。

## 二、地图交互系统

### 2.1 地图探索度系统

每个区域都有独立的探索进度（0-100%），玩家首次进入区域时探索度为0%，通过在区域内移动、发现隐藏地点、与NPC交互、采集资源等方式提升探索度。

**探索度等级划分：**

| 等级 | 探索度 | 状态名称 | 解锁内容 |
|------|--------|----------|----------|
| 0 | 0% | 未探索 | 区域在地图上显示为"???" |
| 1 | 1-25% | 初步探索 | 区域名称显示，基本地形可见 |
| 2 | 26-50% | 部分探索 | 主要地标显示，NPC位置标记 |
| 3 | 51-75% | 深入探索 | 隐藏地点提示出现，资源点标记 |
| 4 | 76-99% | 大部分探索 | 所有地标显示，隐藏通道提示 |
| 5 | 100% | 完全探索 | 所有内容解锁，获得"区域达人"称号 |

**探索度获取方式：**

| 行为 | 探索度增量 | 说明 |
|------|-----------|------|
| 首次进入区域 | +5% | 自动获得 |
| 走过区域中50%的格子 | +15% | 自动计算 |
| 发现隐藏地点 | +10%/个 | 山洞、密室、暗道等 |
| 与区域NPC对话 | +2%/NPC | 首次对话 |
| 完成区域任务 | +5%/任务 | 主线和支线均算 |
| 采集区域资源 | +1%/次 | 采药、采矿、伐木等 |
| 击败区域BOSS | +10% | 首次击败 |
| 在区域中待满24小时 | +3% | 游戏内时间 |

### 2.2 隐藏区域发现机制

部分区域包含隐藏地点，需要满足特定条件才能发现。

**隐藏区域发现条件类型：**

| 类型 | 说明 | 示例 |
|------|------|------|
| 探索触发 | 探索度达到阈值后自动发现 | 探索度>80%发现古墓入口 |
| 物品触发 | 持有特定物品后可进入 | 持有"八卦罗盘"可进入八卦门密室 |
| 时间触发 | 特定时间段出现 | 子时(23:00-01:00)洛阳城门密道开启 |
| 天气触发 | 特定天气下出现 | 暴风雨时华山长空栈道尽头出现山洞 |
| 任务触发 | 完成特定任务后解锁 | 完成"秦陵探秘"任务后可进入地下宫殿 |
| 武功触发 | 使用特定武功破坏障碍 | 用"大力金刚掌"击碎石壁发现密室 |
| 声望触发 | 声望达到要求后NPC告知 | 声望>50时客栈老板告知隐藏商人位置 |
| 组合触发 | 多个条件同时满足 | 探索度>60% + 持有"夜明珠" + 子时 = 发现秦陵地宫 |

### 2.3 快速旅行系统

| 规则 | 说明 |

| 消耗 | 消耗一定内力（距离越远消耗越大） |
| 限制 | 战斗中、被追击中、在门派禁地中不可使用 |
| 特殊传送 | 某些区域有传送阵，可无视距离直接传送 |
