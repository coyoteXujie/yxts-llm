## 1. 技术栈概述

### 1.1 核心技术栈

本项目是一个基于 Python 的武侠 RPG 游戏，采用以下核心技术栈：

| 层级 | 技术 | 用途 | 版本要求 |
|------|------|------|----------|
| **运行时** | Python | 主开发语言 | 3.10+ |
| **游戏引擎** | Arcade | 窗口管理、输入处理、2D渲染、主循环 | 2.x |
| **图像处理** | Pillow (PIL) | 程序化纹理生成、水墨风格叠加、精灵合成 | 9.x |
| **数学计算** | math / random | 坐标变换、概率计算、动画插值 | 标准库 |
| **LLM集成** | llm_client.py | NPC智能对话、奇遇生成（可选模块） | - |
| **存档系统** | json | 游戏状态序列化存档 | 标准库 |

### 1.2 设计哲学

**程序化生成优先**
- 所有视觉资产（地形、角色、UI）均通过代码程序化生成
- 不依赖外部图片文件，降低分发复杂度
- 运行时动态生成，支持无限变体

**水墨国风美学**
- 通过 `ink_style_enabled` 开关可切换水墨画风格渲染
- 支持宣纸底纹、墨迹晕染、飞白效果
- 色调渲染支持多种预设

**轻量状态管理**
- 使用 `@dataclass` 替代重量级 ECS 框架
- 类型提示（type hints）全覆盖，便于维护
- 无外部依赖状态管理库

**模块化系统**
- 各子系统（战斗、经济、修炼、剧情）独立封装
- 通过 `link_player()` 统一联动
- 清晰的接口定义，便于扩展

```python
@dataclass
class Position:
    """位置"""
    x: float = 0.0
    y: float = 0.0
    
    def distance_to(self, other: 'Position') -> float:
        """计算到另一个位置的距离"""
        return math.sqrt((self.x - other.x)**2 + (self.y - other.y)**2)
    
    def __add__(self, other: 'Position') -> 'Position':
        """位置相加"""
        return Position(self.x + other.x, self.y + other.y)

@dataclass
class Skill:
    """技能"""
    id: str
    name: str
    type: SkillType
    level: int = 1
    exp: int = 0
    damage: int = 0
    accuracy: float = 0.8
    
    def get_exp_for_next_level(self) -> int:
        """获取升级所需经验"""
        return int(100 * (1.5 ** self.level))
    
    def add_exp(self, amount: int) -> bool:
        """添加经验，返回是否升级"""
        self.exp += amount
        needed = self.get_exp_for_next_level()
        if self.exp >= needed:
            self.level += 1
            self.exp -= needed
            return True
        return False
```

### 2.2 渲染层文件详解

#### 2.2.1 tile_renderer.py - 地形渲染

**文件路径**：`src/core/render/tile_renderer.py`

**职责说明**：
- 渲染游戏世界的地形tile
- 管理chunk缓存（LRU算法）
- 视口裁剪优化
- 支持tile动画（水面流动）

**核心功能**：

```python
class TileRenderer:
    """地形渲染器"""
    
    def __init__(self):
        self._chunk_cache = {}      # chunk缓存
        self._chunk_size = 10       # 每个chunk为10x10 tile
        self._texture_cache = {}    # tile纹理缓存
    
    def draw_map(self, map_data, camera_x, camera_y):
        """
        绘制地图
        
        流程：
        1. 计算可见tile范围
        2. 按chunk批量渲染
        3. 应用chunk缓存
        """
        # 视口范围计算
        start_col = max(0, int((camera_x - SW/2) / TS))
        end_col = min(map_width, int((camera_x + SW/2) / TS) + 1)
        start_row = max(0, int((camera_y - SH/2) / TS))
        end_row = min(map_height, int((camera_y + SH/2) / TS) + 1)
        
        # 遍历可见tile
        for row in range(start_row, end_row):
            for col in range(start_col, end_col):
                tid = map_data[row][col]
                screen_x = col * TS - camera_x + SW/2
                screen_y = row * TS - camera_y + SH/2
                self._draw_tile(tid, screen_x, screen_y)
    
    def _get_chunk_key(self, col: int, row: int) -> tuple:
        """计算chunk键"""
        return (col // self._chunk_size, row // self._chunk_size)
    
    def _render_chunk(self, chunk_key: tuple) -> arcade.Texture:
        """
        渲染单个chunk
        
        使用PIL生成chunk纹理
        """
        if chunk_key in self._chunk_cache:
            return self._chunk_cache[chunk_key]
        
        # 创建PIL图像
        img = Image.new('RGB', (self._chunk_size * TS, self._chunk_size * TS))
        draw = ImageDraw.Draw(img)
        
        # 绘制chunk内所有tile
        for r in range(self._chunk_size):
            for c in range(self._chunk_size):
                tid = self._get_tile_id(chunk_key, r, c)
                tile_img = self._generate_tile_image(tid)
                img.paste(tile_img, (c * TS, r * TS))
        
        # 转换为Texture并缓存
        texture = self._pil_to_texture(img)
        self._chunk_cache[chunk_key] = texture
        return texture
    
    def invalidate_cache(self):
        """使缓存失效"""
        self._chunk_cache.clear()
        self._texture_cache.clear()
```

```python
class CombatPhase(Enum):
    """战斗阶段"""
    READY = auto()           # 准备阶段
    ROUND_START = auto()     # 回合开始
    SELECT_ACTION = auto()   # 选择行动
    EXECUTE = auto()         # 执行行动
    ROUND_END = auto()       # 回合结束
    VICTORY = auto()         # 胜利
    DEFEAT = auto()          # 失败

class HybridCombatSystem:
    """混合战斗系统（半即时探索 + 回合制战斗）"""
    
    def __init__(self):
        self.phase = CombatPhase.READY
        self.player: Combatant = None
        self.enemies: List[Combatant] = []
        self.round = 0
        self.combat_log: List[str] = []
        self.animation_timer = 0.0
    
    def start_combat(self, player: Player, enemies: List[NPC]):
        """
        开始战斗
        
        流程：
        1. 创建Combatant快照
        2. 计算先手
        3. 初始化战斗状态
        """
        self.player = self._create_combatant(player)
        self.enemies = [self._create_combatant(e) for e in enemies]
        
        # 速度决定先手
        if self.player.speed >= max(e.speed for e in self.enemies):
            self.phase = CombatPhase.SELECT_ACTION
        else:
            self.phase = CombatPhase.ENEMY_TURN
    
    def execute_action(self, action: str, target_index: int = None):
        """
        执行行动
        
        支持的行动类型：
        - "attack": 普通攻击
        - "skill": 使用技能
        - "defend": 防御
        - "item": 使用物品
        - "flee": 逃跑
        """
        if action == "attack":
            damage = self._calculate_damage(self.player, self.enemies[target_index])
            self.enemies[target_index].hp -= damage
            self.combat_log.append(f"玩家造成 {damage} 点伤害！")
        
        elif action == "skill":
            skill = self.player.skills[target_index]
            damage = self._calculate_skill_damage(skill, self.enemies[target_index])
            self.player.mp -= skill.mp_cost
            self.enemies[target_index].hp -= damage
            self.combat_log.append(f"玩家使用「{skill.name}」！")
        
        elif action == "defend":
            self.player.is_defending = True
            self.combat_log.append("玩家进入防御姿态！")
    
    def _calculate_damage(self, attacker: Combatant, defender: Combatant) -> int:
        """
        计算普通攻击伤害
        
        公式：
        damage = (attacker.attack - defender.defense * 0.5) * random(0.8, 1.2)
        """
        base = attacker.attack - defender.defense * 0.5
        variance = random.uniform(0.8, 1.2)
        return max(1, int(base * variance))
```

```python
    def buy_item(self, item_id: str, quantity: int = 1) -> bool:
        """
        购买物品
        
        流程：
        1. 获取当前价格（含波动）
        2. 检查玩家金币
        3. 扣除金币
        4. 添加物品到背包
        5. 触发事件
        """
        price = self.get_current_price(item_id) * quantity
        if self.player.money < price:
            return False
        
        self.player.money -= price
        self.player.inventory.add(item_id, quantity)
        
        dispatch(EventType.ITEM_PURCHASED, {"item_id": item_id, "price": price})
        return True
    
    def sell_item(self, item_id: str, quantity: int = 1) -> bool:
        """
        出售物品
        
        收购价为售价的50%
        """
        if not self.player.inventory.contains(item_id, quantity):
            return False
        
        base_price = ITEMS_DB[item_id].price
        sell_price = int(base_price * 0.5 * quantity)
        
        self.player.inventory.remove(item_id, quantity)
        self.player.money += sell_price
        
        dispatch(EventType.ITEM_SOLD, {"item_id": item_id, "price": sell_price})
        return True
    
    def get_current_price(self, item_id: str) -> int:
        """
        获取当前价格
        
        价格 = 基础价格 × 供需系数 × 时间系数 × 区域系数
        """
        base_price = ITEMS_DB[item_id].price
        demand_factor = self.price_fluctuation.get(item_id, 1.0)
        time_factor = self._get_time_factor()
        region_factor = self._get_region_factor()
        
        return int(base_price * demand_factor * time_factor * region_factor)
```

```python
class ReputationSystem:
    """声望系统"""
    
    def change_daode(self, delta: int, reason: str = ""):
        """
        改变道德值
        
        边界情况处理：
        1. 上限检查（道德值上限）
        2. 下限检查（道德值下限）
        3. 阈值跨越处理
        4. 连锁反应处理
        """
        old_daode = self.player.daode
        new_daode = old_daode + delta
        
        # 上限检查
        MAX_DAODE = 100
        if new_daode > MAX_DAODE:
            new_daode = MAX_DAODE
        
        # 下限检查
        MIN_DAODE = -100
        if new_daode < MIN_DAODE:
            new_daode = MIN_DAODE
        
        self.player.daode = new_daode
        
        # 阈值跨越处理
        self._check_threshold_cross(old_daode, new_daode)
        
        # 触发事件
        dispatch(EventType.DAODE_CHANGED, {
            "old": old_daode,
            "new": new_daode,
            "delta": delta,
            "reason": reason
        })
    
    def _check_threshold_cross(self, old: int, new: int):
        """
        检查阈值跨越
        
        道德等级阈值：
        - 正道大侠：daode >= 80
        - 正道侠士：daode >= 50
        - 中立：-30 < daode < 50
        - 邪道人士：daode <= -30
        - 邪魔外道：daode <= -60
        """
        thresholds = [
            (80, "正道大侠"),
            (50, "正道侠士"),
            (-30, "中立"),
            (-60, "邪道人士"),
            (float('-inf'), "邪魔外道")
        ]
        
        for threshold, title in thresholds:
            if old < threshold <= new:
                self._on_become(title, positive=True)
            elif old > threshold >= new:
                self._on_become(title, positive=False)
    
    def _on_become(self, title: str, positive: bool):
        """成为新称号时的处理"""
        self.player.reputation_title = title
        
        if positive:
            # 触发庆祝事件
            dispatch(EventType.REPUTATION_UP, {"title": title})
            # NPC态度变化
            self._update_npc_attitudes()
        else:
            # 触发警告事件
            dispatch(EventType.REPUTATION_DOWN, {"title": title})
```

**LRU缓存实现**：

```python
from collections import OrderedDict
import weakref

class LRUCache:
    """LRU缓存实现"""
    
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()
    
    def get(self, key):
        """获取缓存项"""
        if key in self.cache:
            # 移到末尾（最近使用）
            self.cache.move_to_end(key)
            return self.cache[key]
        return None
    
    def put(self, key, value):
        """放入缓存"""
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        
        # 超过容量，移除最旧的
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)

class ChunkCache:
    """Chunk纹理缓存"""
    
    def __init__(self, max_chunks: int = 100):
        self.max_chunks = max_chunks
        self.cache = LRUCache(max_chunks)
        self._render_counts = {}  # 渲染计数
    
    def get_chunk(self, chunk_key: tuple) -> Optional[arcade.Texture]:
        """获取chunk纹理"""
        texture = self.cache.get(chunk_key)
        if texture:
            self._render_counts[chunk_key] = self._render_counts.get(chunk_key, 0) + 1
        return texture
    
    def put_chunk(self, chunk_key: tuple, texture: arcade.Texture):
        """缓存chunk纹理"""
        self.cache.put(chunk_key, texture)
    
    def invalidate(self, chunk_key: tuple = None):
        """使缓存失效"""
        if chunk_key:
            if chunk_key in self.cache.cache:
                del self.cache.cache[chunk_key]
        else:
            self.cache.cache.clear()
    
    def get_stats(self) -> dict:
        """获取缓存统计"""
        return {
            "cached_chunks": len(self.cache.cache),
            "render_counts": self._render_counts,
            "hit_rate": self._calculate_hit_rate()
        }
    
    def _calculate_hit_rate(self) -> float:
        """计算命中率"""
        total = sum(self._render_counts.values())
        if total == 0:
            return 0.0
        cached = sum(1 for k, v in self._render_counts.items() 
                    if k in self.cache.cache)
        return cached / total
```

**内存管理**：

```python
class ChunkMemoryManager:
    """Chunk内存管理器"""
    
    def __init__(self, max_memory_mb: float = 256):
        self.max_memory = max_memory_mb * 1024 * 1024
        self.current_memory = 0
        self.chunk_cache = ChunkCache()
    
    def estimate_chunk_size(self, chunk_key: tuple) -> int:
        """
        估算chunk大小
        
        假设每个chunk为10x10 tile，每个tile 64x64 RGBA
        chunk_size = 10 * 10 * 64 * 64 * 4 bytes ≈ 1.6MB
        """
        tile_count = 10 * 10
        pixels_per_tile = 64 * 64
        bytes_per_pixel = 4
        return tile_count * pixels_per_tile * bytes_per_pixel
    
    def can_allocate(self, chunk_key: tuple) -> bool:
        """检查是否可以分配"""
        chunk_size = self.estimate_chunk_size(chunk_key)
        return (self.current_memory + chunk_size) <= self.max_memory
    
    def allocate(self, chunk_key: tuple, texture: arcade.Texture):
        """分配chunk内存"""
        chunk_size = self.estimate_chunk_size(chunk_key)
        
        # 如果内存不足，释放最少使用的chunk
        while (self.current_memory + chunk_size) > self.max_memory:
            if not self._evict_lru():
                break  # 无法释放更多
        
        self.chunk_cache.put_chunk(chunk_key, texture)
        self.current_memory += chunk_size
    
    def _evict_lru(self) -> bool:
        """驱逐最少使用的chunk"""
        # 找到最少使用的chunk
        if not self.chunk_cache.cache.cache:
            return False
        
        # OrderedDict.popitem(last=False) 移除第一个（最旧）
        lru_key = next(iter(self.chunk_cache.cache.cache))
        
        chunk_size = self.estimate_chunk_size(lru_key)
        del self.chunk_cache.cache.cache[lru_key]
        self.current_memory -= chunk_size
        
        return True
```

### 6.2 水墨纹理叠加原理

```python
class InkTextureGenerator:
    """水墨纹理生成器"""
    
    def __init__(self):
        self.paper_texture = None
        self.ink_textures = {}
    
    def generate_paper_texture(self, width: int, height: int) -> Image.Image:
        """
        生成宣纸底纹
        
        流程：
        1. 创建基础米色背景
        2. 添加噪点模拟纸张纹理
        3. 应用轻微模糊
        """
        img = Image.new('RGB', (width, height), (245, 240, 230))
        draw = ImageDraw.Draw(img)
        
        # 添加噪点
        rng = random.Random(42)
        for _ in range(width * height // 100):
            x = rng.randint(0, width - 1)
            y = rng.randint(0, height - 1)
            noise = rng.randint(-5, 5)
            color = (
                max(0, min(255, 245 + noise)),
                max(0, min(255, 240 + noise)),
                max(0, min(255, 230 + noise))
            )
            draw.point((x, y), fill=color)
        
        # 轻微模糊
        img = img.filter(ImageFilter.GaussianBlur(radius=0.5))
        
        return img
    
    def generate_ink_texture(self, width: int, height: int, 
                            ink_type: str = "wash") -> Image.Image:
        """
        生成墨迹纹理
        
        ink_type:
        - "wash": 水墨晕染效果
        - "stroke": 毛笔笔触效果
        - "splash": 墨迹飞溅效果
        """
        img = Image.new('RGBA', (width, height), (0, 0, 0, 0))
        draw = ImageDraw.Draw(img)
        
        if ink_type == "wash":
            # 水墨晕染
            rng = random.Random(42)
            for _ in range(20):
                x = rng.randint(0, width)
                y = rng.randint(0, height)
                r = rng.randint(10, 50)
                alpha = rng.randint(10, 40)
                draw.ellipse([x-r, y-r, x+r, y+r], 
                           fill=(30, 30, 30, alpha))
            
            img = img.filter(ImageFilter.GaussianBlur(radius=3))
        
        elif ink_type == "stroke":
            # 毛笔笔触
            rng = random.Random(42)
            for _ in range(10):
                x1 = rng.randint(0, width)
                y1 = rng.randint(0, height)
                x2 = x1 + rng.randint(-50, 50)
                y2 = y1 + rng.randint(-50, 50)
                width_stroke = rng.randint(2, 8)
                alpha = rng.randint(80, 150)
                draw.line([x1, y1, x2, y2], 
                         fill=(30, 30, 30, alpha),
                         width=width_stroke)
        
        return img
    
    def apply_ink_style(self, source: Image.Image, 
                       opacity: float = 0.35) -> Image.Image:
        """
        应用水墨风格到图像
        
        参数：
        - source: 源图像
        - opacity: 墨迹叠加透明度
        """
        # 生成宣纸底纹
        paper = self.generate_paper_texture(source.width, source.height)
        
        # 生成墨迹
        ink = self.generate_ink_texture(source.width, source.height)
        
        # 合并
        result = Image.new('RGB', source.size)
        source_rgba = source.convert('RGBA')
        
        # 调整墨迹透明度
        ink_alpha = Image.new('RGBA', ink.size, (0, 0, 0, int(255 * opacity)))
        ink_composite = Image.alpha_composite(ink, ink_alpha)
        
        # 叠加
        result = Image.blend(source_rgba.convert('RGB'), ink_composite.convert('RGB'), opacity)
        
        return result
```

```python
        添加矩形到批处理队列
        
        不会立即渲染，而是累积后一次性渲染
        """

            "x": x, "y": y,
            "width": width, "height": height,
            "color": color,
            "draw_type": draw_type
        })
    
    def add_text(self, text: str, x: float, y: float, 
                color: tuple, size: int = 12):
        """添加文本到批处理队列"""
        self._pending_texts.append({
            "text": text,
            "x": x, "y": y,
            "color": color,
            "size": size
        })
    
    def render(self):
        """
        执行批量渲染
        
        将所有累积的绘制命令合并执行
        """
        # 批量绘制矩形
        for rect in self._pending_rects:
            if rect["draw_type"] == "filled":
                arcade.draw_rectangle_filled(
                    rect["x"], rect["y"],
                    rect["width"], rect["height"],
                    rect["color"]
                )
            else:
                arcade.draw_rectangle_outline(
                    rect["x"], rect["y"],
                    rect["width"], rect["height"],
                    rect["color"]
                )
        
        # 批量绘制文本
        for text_data in self._pending_texts:
            arcade.draw_text(
                text_data["text"],
                text_data["x"], text_data["y"],
                text_data["color"],
                text_data["size"]
            )
        
        # 清空队列
```

```python
        self.animations.append({
            "type": "attack",
            "start_pos": attacker_pos,
            "end_pos": target_pos,
            "progress": 0.0,
            "duration": 0.3,
            "skill_name": skill_name
        })

                              is_critical: bool = False,
                              is_heal: bool = False):
        """触发伤害数字"""
        self.damage_numbers.append({
            "damage": damage,
            "x": pos[0],
            "y": pos[1],
            "start_y": pos[1],
            "is_critical": is_critical,
            "is_heal": is_heal,
            "progress": 0.0,
            "duration": 1.2
        })
    
    def trigger_screen_shake(self, intensity: int = 10):
        """触发屏幕震动"""
        import random
        self.screen_shake = (
            random.randint(-intensity, intensity),
            random.randint(-intensity, intensity)
        )
    
    def update(self, delta_time: float):
        """更新所有动画"""
        # 更新攻击动画
        for anim in self.animations[:]:
            anim["progress"] += delta_time / anim["duration"]
            if anim["progress"] >= 1.0:
                self.animations.remove(anim)
        
        # 更新伤害数字
        for dmg in self.damage_numbers[:]:
            dmg["progress"] += delta_time / dmg["duration"]
            dmg["y"] = dmg["start_y"] + dmg["progress"] * 60  # 上升60像素
            if dmg["progress"] >= 1.0:
                self.damage_numbers.remove(dmg)
        
        # 更新屏幕震动（衰减）
        if self.screen_shake != (0, 0):
            self.screen_shake = (
                int(self.screen_shake[0] * 0.9),
                int(self.screen_shake[1] * 0.9)
            )
            if abs(self.screen_shake[0]) < 1 and abs(self.screen_shake[1]) < 1:
                self.screen_shake = (0, 0)
        
        # 更新闪光

            self.flash_alpha = max(0, self.flash_alpha - 10)
```

### 7.4 配置文件的热重载

```python
import json
import threading

class ConfigReloader:
    """配置热重载系统"""
    
    def __init__(self, config_path: str):
        self.config_path = config_path
        self.config = {}
        self._last_mtime = 0
        self._lock = threading.Lock()
        self._callback = None
        
        # 初始加载
        self.reload()
    
    def reload(self):
        """重新加载配置"""
        import os
        if not os.path.exists(self.config_path):
            return
        
        mtime = os.path.getmtime(self.config_path)
        if mtime <= self._last_mtime:
            return  # 文件未修改
        
        with self._lock:
            try:
                with open(self.config_path, 'r', encoding='utf-8') as f:
                    new_config = json.load(f)
                
                # 合并配置
                self._merge_config(new_config)
                self._last_mtime = mtime
                
                # 触发回调
                if self._callback:
                    self._callback(self.config)
                    
            except Exception as e:
                print(f"配置加载失败: {e}")
    
    def _merge_config(self, new_config: dict):
        """合并配置（保留未修改的项）"""
        def merge_dict(d1, d2):
            for key, value in d2.items():
                if key in d1 and isinstance(d1[key], dict) and isinstance(value, dict):
                    merge_dict(d1[key], value)
                else:
                    d1[key] = value
        
        merge_dict(self.config, new_config)
    
    def get(self, key: str, default=None):
        """获取配置值"""
        with self._lock:
            return self.config.get(key, default)
    
    def watch(self, callback):
        """注册配置变更回调"""
        self._callback = callback
    
    def start_watching(self, interval: float = 1.0):
        """开始监控配置文件变化"""
        import threading
        
        def watch_loop():
            while True:
                self.reload()
                threading.Event().wait(interval)
        
        thread = threading.Thread(target=watch_loop, daemon=True)
        thread.start()
```

```python
class WorldRegionManager:
    """世界区域管理器"""
    
    def __init__(self):
        # 区域缓存
        self._loaded_regions: Dict[str, Map] = {}
        self._region_metadata: Dict[str, RegionMetadata] = {}
        
        # 缓存策略
        self.max_cached_regions = 9  # 最多缓存9个区域
        self.current_region: str = None
        
        # LRU追踪
        self._access_order: List[str] = []
    
    def load_region(self, region_id: str) -> Map:
        """
        加载区域
        
        策略：
        1. 检查是否已加载
        2. 检查缓存是否已满
        3. 按需加载/替换
        """
        # 已加载
        if region_id in self._loaded_regions:
            self._mark_access(region_id)
            return self._loaded_regions[region_id]
        
        # 缓存已满，替换最少使用的
        if len(self._loaded_regions) >= self.max_cached_regions:
            self._evict_lru()
        
        # 加载新区
        region = self._load_from_disk(region_id)
        self._loaded_regions[region_id] = region
        self._mark_access(region_id)
        
        return region
    
    def _mark_access(self, region_id: str):
        """标记访问（更新LRU顺序）"""
        if region_id in self._access_order:
            self._access_order.remove(region_id)
        self._access_order.append(region_id)
    
    def _evict_lru(self):
        """驱逐最少使用的区域"""
        if not self._access_order:
            return
        
        lru_id = self._access_order[0]
        
        # 保存修改
        self._save_to_disk(lru_id, self._loaded_regions[lru_id])
        
        # 移除
        del self._loaded_regions[lru_id]
        self._access_order.pop(0)
    
    def get_adjacent_regions(self, region_id: str) -> List[str]:
        """获取相邻区域列表"""
        metadata = self._region_metadata.get(region_id)
        if metadata:
            return metadata.adjacent_regions
        return []
    
    def preload_adjacent(self, region_id: str):
        """
        预加载相邻区域
        
        用于：进入新区域时预加载周围区域
        """
        adjacent = self.get_adjacent_regions(region_id)
        for adj_id in adjacent:
            if adj_id not in self._loaded_regions:
                self.load_region(adj_id)
```

```python
class ChunkCacheOptimizer:
    """Chunk缓存优化器"""
    
    def __init__(self):
        self.cache = ChunkCache(capacity=100)
        self.hit_count = 0
        self.miss_count = 0
        self._predict_queue = []
    
    def record_hit(self, chunk_key: tuple):
        """记录缓存命中"""
        self.hit_count += 1
        # 分析访问模式
        self._analyze_access_pattern(chunk_key)
    
    def record_miss(self, chunk_key: tuple):
        """记录缓存未命中"""
        self.miss_count += 1
    
    def get_hit_rate(self) -> float:
        """获取命中率"""
        total = self.hit_count + self.miss_count
        if total == 0:
            return 0.0
        return self.hit_count / total
    
    def _analyze_access_pattern(self, chunk_key: tuple):
        """
        分析访问模式
        
        发现规律：玩家通常沿直线移动，相邻chunk会被连续访问
        优化策略：预测下一个chunk并预加载
        """
        self._predict_queue.append(chunk_key)
        if len(self._predict_queue) > 5:
            self._predict_queue.pop(0)
    
    def predict_next_chunks(self, player_dir: Tuple[float, float],
                           player_chunk: tuple) -> List[tuple]:
        """
        预测下一个要访问的chunk
        
        基于：
        1. 玩家移动方向
        2. 历史访问模式
        3. 相邻chunk优先
        """
        predictions = []
        
        # 基于方向的预测
        dx, dy = player_dir
        if abs(dx) > abs(dy):  # 水平移动为主
            # 预测左右chunk
            predictions.append((player_chunk[0] + 1, player_chunk[1]))
            predictions.append((player_chunk[0] - 1, player_chunk[1]))
        else:  # 垂直移动为主
            # 预测上下chunk
            predictions.append((player_chunk[0], player_chunk[1] + 1))
            predictions.append((player_chunk[0], player_chunk[1] - 1))
        
        # 基于历史的预测
        if len(self._predict_queue) >= 2:
            # 速度趋势
            last = self._predict_queue[-1]
            prev = self._predict_queue[-2]
            velocity = (last[0] - prev[0], last[1] - prev[1])
            if velocity != (0, 0):
                next_chunk = (last[0] + velocity[0], last[1] + velocity[1])
                predictions.append(next_chunk)
        
        return predictions
    
    def prefetch(self, chunks: List[tuple]):
        """
        预加载chunk
        
        异步执行，不阻塞主线程
        """
        for chunk_key in chunks:
            if chunk_key not in self.cache.cache.cache:
                texture = self._render_chunk(chunk_key)
                self.cache.put_chunk(chunk_key, texture)
```

```python
class ViewportCuller:
    """视口裁剪器"""
    
    def __init__(self):
        self.viewport_width = GAME_CONFIG["screen_width"]
        self.viewport_height = GAME_CONFIG["screen_height"]
        self.tile_size = GAME_CONFIG["tile_size"]
    
    def get_visible_tiles(self, camera_x: float, camera_y: float,
                        map_width: int, map_height: int,
                        margin: float = 1.0) -> Tuple[int, int, int, int]:
        """
        获取可见tile范围
        
        参数：
        - camera_x, camera_y: 相机中心位置
        - map_width, map_height: 地图尺寸
        - margin: 扩展边距（用于平滑滚动）
        
        返回：(start_col, end_col, start_row, end_row)
        """
        # 计算视口范围
        half_w = self.viewport_width / 2 + margin * self.tile_size
        half_h = self.viewport_height / 2 + margin * self.tile_size
        
        # 转换为tile坐标
        start_col = max(0, int((camera_x - half_w) / self.tile_size))
        end_col = min(map_width, int((camera_x + half_w) / self.tile_size) + 1)
        start_row = max(0, int((camera_y - half_h) / self.tile_size))
        end_row = min(map_height, int((camera_y + half_h) / self.tile_size) + 1)
        
        return start_col, end_col, start_row, end_row
    
    def count_visible_tiles(self, camera_x: float, camera_y: float,
                           map_width: int, map_height: int) -> int:
        """计算可见tile数量"""
        start_col, end_col, start_row, end_row = self.get_visible_tiles(
            camera_x, camera_y, map_width, map_height
        )
        return (end_col - start_col) * (end_row - start_row)
    
    def cull_entities(self, entities: List['Entity'],
                     camera_x: float, camera_y: float) -> List['Entity']:
        """
        裁剪屏幕外的实体
        
        参数：
        - entities: 实体列表
        - camera_x, camera_y: 相机位置
        
        返回：可见实体列表
        """
        half_w = self.viewport_width / 2
        half_h = self.viewport_height / 2
        margin = 100  # 额外边距
        
        visible = []
        for entity in entities:
            ex, ey = entity.x, entity.y
            
            # 检查是否在视口内（带边距）
```

```python
class ObjectPool(Generic[T]):
    """通用对象池"""
    
    def __init__(self, factory: Callable[[], T], initial_size: int = 10):
        self._factory = factory
        self._pool: List[T] = []
        self._in_use: Set[int] = set()
        
        # 预创建对象
        for _ in range(initial_size):
            self._pool.append(factory())
    
    def acquire(self) -> T:
        """获取对象"""
        if self._pool:
            obj = self._pool.pop()
        else:
            obj = self._factory()
        
        self._in_use.add(id(obj))
        return obj
    
    def release(self, obj: T):
        """归还对象"""
        obj_id = id(obj)
        if obj_id in self._in_use:
            self._in_use.remove(obj_id)
            
            # 重置对象状态
            if hasattr(obj, 'reset'):
                obj.reset()
            
            self._pool.append(obj)
    
    def __enter__(self):
        """上下文管理器支持"""
        return self.acquire()
    
    def __exit__(self, exc_type, exc_val, exc_tb):
        """上下文管理器支持"""
        pass  # 需要外部传入对象
```

```python
class MemoryLeakProtector:

    def __init__(self):
        # 弱引用追踪
        self._weak_refs: Dict[str, weakref.ref] = {}
        
        # 引用计数追踪

        """
        注册弱引用
        
        用于：事件监听器、回调、观察者
        防止循环引用导致的内存泄漏
        """
        self._weak_refs[key] = weakref.ref(obj)
    
    def check_memory(self, current_time: float):
        """
        检查内存使用
        
        定时触发GC
        """
        if current_time - self._last_gc_time < self._gc_interval:
            return
        
        self._last_gc_time = current_time
        
        # 强制GC
        gc.collect()
        
        # 检查内存
        import psutil
        process = psutil.Process()
        memory_mb = process.memory_info().rss / 1024 / 1024
        
        if memory_mb > self._memory_threshold_mb:
            # 内存警告
            dispatch(EventType.MEMORY_WARNING, {
                "memory_mb": memory_mb,
                "threshold_mb": self._memory_threshold_mb
            })
    
    def find_circular_refs(self) -> Set[frozenset]:
        """
        查找循环引用
        
        使用gc.get_referrers分析
        """
        circular = set()
        
        # 获取所有对象
        objs = gc.get_objects()
        
        for obj in objs:
            if not hasattr(obj, '__dict__'):
                continue
            
            # 分析引用关系
            refs = gc.get_referrers(obj)
            for ref in refs:
                if ref is not obj and isinstance(ref, dict):
                    # 检查是否形成循环
                    if obj in gc.get_referrers(ref):
                        circular.add(frozenset([id(obj), id(ref)]))
        
        return circular
```

```python
class PluginSystem:
    """
    插件系统架构
    
    设计原则：
    - 插件独立加载/卸载
    - 插件通过接口与主程序交互
    - 支持热重载
    """
    
    def __init__(self):
        self.plugins: Dict[str, Plugin] = {}
        self.hooks: Dict[str, List[Callable]] = {}
        self.plugin_dirs = ["plugins/", "user_plugins/"]
    
    def load_plugin(self, plugin_name: str) -> bool:
        """
        加载插件
        
        插件结构：
        plugins/
        ├── plugin_name/
        │   ├── __init__.py
        │   ├── plugin.py      # 主插件类
        │   ├── hooks.py       # 钩子函数
        │   └── data/          # 插件数据
        """
        import importlib.util
        
        # 查找插件
        plugin_path = self._find_plugin(plugin_name)
        if not plugin_path:
            return False
        
        # 加载模块
        spec = importlib.util.spec_from_file_location(
            f"plugins.{plugin_name}", 
            plugin_path
        )
        module = importlib.util.module_from_spec(spec)
        spec.loader.exec_module(module)
        
        # 创建插件实例
        plugin_class = getattr(module, "Plugin", None)
        if not plugin_class:
            return False
        
        plugin = plugin_class()
        plugin.on_load()
        
        # 注册钩子
        self._register_hooks(plugin)
        
        # 保存
        self.plugins[plugin_name] = plugin
        
        return True
    
    def unload_plugin(self, plugin_name: str) -> bool:
        """卸载插件"""
        plugin = self.plugins.get(plugin_name)
        if not plugin:
            return False
        
        # 调用卸载钩子
        plugin.on_unload()
        
        # 注销钩子
        self._unregister_hooks(plugin)
        
        # 移除
        del self.plugins[plugin_name]
        
        return True
    
    def hook(self, event_name: str):
        """
        装饰器：注册事件钩子
        
        用法：
        @plugin.hook("on_player_move")
        def on_player_move(player, new_pos):
            print(f"玩家移动到 {new_pos}")
        """
        def decorator(func: Callable):
            if event_name not in self.hooks:
                self.hooks[event_name] = []
            self.hooks[event_name].append(func)
            return func
        return decorator
    
    def trigger(self, event_name: str, *args, **kwargs):
        """
        触发事件钩子
        
        所有注册的钩子函数会被依次调用
        """
        for hook in self.hooks.get(event_name, []):
            try:
                hook(*args, **kwargs)
            except Exception as e:
                print(f"Hook error in {event_name}: {e}")
```
