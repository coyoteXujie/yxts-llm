# 侠影江湖 - 故事、任务与冒险设计文档

---

## 一、主线故事（5章+3结局）

### 第一章：初入江湖

**章节名称**：少年侠影

**开启条件**：游戏开始，主角从平安镇出发

**故事背景**：
主角出生在江南水乡平安镇，父亲曾是江湖上威名赫赫的大侠「凌云剑」李慕白，十年前因卷入一场江湖阴谋而神秘失踪。母亲带着年幼的主角隐居平安镇，过着平静的生活。然而，平静的日子在主角十六岁这年被打破——母亲身患奇疾，遍请名医束手无策，只有传说中的「还魂草」才能救治。为救母亲，主角踏上了寻找还魂草的征程，从此踏入了波谲云诡的江湖。

**详细剧情流程**：

**起因**：
- 清晨，主角在平安镇家中醒来，发现母亲病情加重
- 郎中告知只有传说中的还魂草能救母亲，但还魂草生长在凶险的黑风岭
- 主角下定决心，告别母亲，踏上寻找还魂草之路

**经过**：
1. **初遇麻烦**：主角刚出平安镇，就遇到几个地痞流氓骚扰一位少女（林月如）
   - 分支选择：
     - 选择A：出手相助（获得林月如好感，开启后续剧情）
     - 选择B：绕道而行（林月如自己解决，但好感下降，后续剧情略有不同）
2. **黑风岭探险**：主角到达黑风岭，遇到各种野兽和山贼
   - 关键战斗：黑风寨寨主「黑风煞」
3. **获得还魂草**：在黑风岭深处找到还魂草，但同时发现一个神秘令牌
4. **返回平安镇**：主角返回平安镇，用还魂草救了母亲
5. **神秘访客**：当晚，一位神秘老者来访，告知主角令牌的来历——这是「影门」的信物
6. **父亲的消息**：老者透露，十年前李慕白的失踪与影门和各大势力都有关系
7. **决定出发**：主角决定离开平安镇，前往江湖寻找父亲的下落

**结果**：
- 母亲得救
- 主角获得影门令牌
- 主角决定踏上寻找父亲的征程
- 开启第二章剧情

**关键NPC**：
1. **李慕雪**（主角母亲）：温柔善良，知晓部分当年的秘密
2. **林月如**：江南林家大小姐，性格泼辣但心地善良
3. **无尘道长**：神秘老者，影门现任掌门，武功深不可测
4. **黑风煞**：黑风寨寨主，无恶不作

**关键战斗**：
1. **地痞流氓×3**：初出茅庐的第一场战斗
2. **野狼×2**：黑风岭的野兽
3. **山贼×5**：黑风寨的喽啰
4. **黑风煞**（BOSS）：黑风寨寨主，使用开山斧

**分支选择**：
1. 是否救助林月如（影响林月如好感度和后续剧情）
2. 是否加入影门（影响后续势力选择）

**奖励**：
- 经验值：500
- 金钱：100两
- 物品：铁剑、粗布衣服、还魂草（任务物品）、影门令牌（关键物品）
- 声望：平安镇+50
- 好感：林月如+20（如选择救助）

### 第二章：风云际会

**章节名称**：江南风波

**开启条件**：完成第一章，主角等级达到10级

**故事背景**：
主角离开平安镇，前往江南第一大城——苏州城。途中听闻江南最近不太平，多家镖局被劫，货物被抢，矛头直指江南最大的帮派「金龙帮」。同时，主角在苏州城遇到了儿时玩伴、如今已是武当派弟子的张天师，以及神秘的医女苏婉清。随着调查深入，主角发现这一切似乎都与十年前父亲的失踪有关。

**详细剧情流程**：

**起因**：
- 主角到达苏州城，发现城内气氛紧张
- 听说多家镖局被劫，金龙帮被指为幕后黑手
- 主角决定调查此事，同时寻找父亲的线索

**经过**：
1. **初到苏州**：主角在苏州城门口遇到张天师，两人相认
2. **镖局惨案**：主角前往「威远镖局」，发现镖局被血洗
3. **线索收集**：在镖局现场发现金龙帮的信物
4. **拜访金龙帮**：主角前往金龙帮总舵，帮主「金霸天」否认此事
5. **遭遇暗杀**：主角在回客栈的路上遭到黑衣人袭击
6. **医女相救**：主角受伤昏迷，被苏婉清救下
7. **发现真相**：苏婉清告知主角，劫镖的并非金龙帮，而是「血影教」的人
8. **深入调查**：主角与张天师、苏婉清一起调查，发现血影教正在策划一个大阴谋
9. **金龙帮被围**：血影教嫁祸金龙帮，各大门派围攻金龙帮
10. **揭露阴谋**：主角在关键时刻拿出证据，揭露血影教的阴谋
11. **大战血影教**：主角与各大门派一起对抗血影教
12. **关键线索**：在血影教据点发现父亲的佩剑——「凌云剑」

**结果**：
- 血影教的阴谋被粉碎
- 金龙帮洗清冤屈
- 主角获得凌云剑
- 得知父亲可能被关押在血影教总坛
- 开启第三章剧情

**关键NPC**：
1. **张天师**：武当派弟子，主角儿时玩伴，正直勇敢
2. **苏婉清**：神秘医女，医术高超，身份不明
3. **金霸天**：金龙帮帮主，性格豪爽，被人嫁祸
4. **血影长老**：血影教高层，武功高强，心狠手辣

**关键战斗**：
1. **黑衣人×3**：苏州城暗杀者
2. **血影教徒×5**：血影教据点守卫
3. **血影护法**（BOSS）：血影教高手
4. **血影长老**（BOSS）：血影教高层

**分支选择**：
1. 是否相信金龙帮（影响金龙帮声望）
2. 是否独自调查还是与张天师合作（影响剧情发展）
3. 凌云剑的处理方式（影响后续剧情）

**奖励**：
- 经验值：2000
- 金钱：500两
- 物品：凌云剑（主角父亲的佩剑）、金疮药×5、解毒丹×3
- 声望：金龙帮+100、武当派+50、苏州城+80
- 好感：张天师+30、苏婉清+40、金霸天+50

### 第三章：剑指中原

**章节名称**：中原逐鹿

**开启条件**：完成第二章，主角等级达到20级

**故事背景**：
主角带着凌云剑，前往中原腹地——洛阳城。一路上，主角发现江湖上各大门派都在暗中行动，似乎在寻找什么东西。同时，主角遇到了少林派的慧明大师，得知十年前的那场阴谋涉及一个更大的秘密——「九龙宝藏」。传说九龙宝藏中藏有绝世武功和富可敌国的财富，父亲正是因为发现了这个秘密而被人陷害。

**详细剧情流程**：

**起因**：
- 主角到达洛阳城，发现城内高手云集
- 慧明大师找到主角，告知九龙宝藏的秘密
- 主角决定继续调查，同时寻找父亲

**经过**：
1. **洛阳盛会**：主角参加洛阳城举办的武林大会
2. **初露锋芒**：主角在武林大会上崭露头角
3. **慧明大师**：慧明大师找到主角，告知九龙宝藏的传说
4. **九龙密卷**：主角获得第一份九龙密卷
5. **各大门派争夺**：各大门派得知密卷消息，纷纷争夺
6. **明教介入**：明教也派出高手，想要夺取密卷
7. **主角遇险**：主角被明教围攻，被一位神秘女子（明教圣女小昭）救下
8. **第二份密卷**：主角在小昭的帮助下获得第二份密卷
9. **密卷解密**：两份密卷合在一起，显示出宝藏的大致位置
10. **父亲的消息**：从密卷中得知，父亲可能在宝藏所在地
11. **前往宝藏**：主角与小昭一起前往宝藏所在地——终南山
12. **终南之战**：在终南山与各大门派、明教展开大战
13. **发现密室**：主角找到宝藏入口，发现父亲留下的书信

**结果**：
- 获得九龙密卷两份
- 得知父亲在西域
- 小昭加入主角队伍
- 开启第四章剧情

**关键NPC**：
1. **慧明大师**：少林派高僧，知晓九龙宝藏的秘密
2. **小昭**：明教圣女，善良纯真，对主角有好感
3. **明教教主**：明教教主，野心勃勃，想要夺取宝藏
4. **各大门派掌门**：武当、峨眉、华山等门派掌门

**关键战斗**：
1. **武林大会对手×3**：武林大会上的对手
2. **明教教徒×5**：明教追兵
3. **明教法王**（BOSS）：明教高手
4. **各大门派高手×3**：终南山争夺者
5. **守墓傀儡×5**：宝藏守卫

**分支选择**：
1. 是否加入明教（影响势力选择）
2. 是否将密卷交给慧明大师（影响声望）
3. 是否相信小昭（影响好感和剧情）

**奖励**：
- 经验值：5000
- 金钱：1000两
- 物品：九龙密卷×2、少林心法（秘籍）、圣火令碎片
- 声望：少林派+100、明教+50（或-100，取决于选择）、洛阳城+100
- 好感：小昭+50、慧明大师+30

### 第四章：西域迷踪

**章节名称**：大漠孤烟

**开启条件**：完成第三章，主角等级达到30级

**故事背景**：
主角根据父亲书信的指引，前往西域。西域是一个神秘的地方，这里有各种各样的江湖势力——天山派、魔教、西域三十六国等等。主角在这里遇到了更多的挑战，也发现了更大的阴谋——原来血影教的背后还有更大的势力，而父亲就在这个势力的掌控之中。

**详细剧情流程**：

**起因**：
- 主角到达西域第一大城——敦煌
- 发现西域局势紧张，各势力之间剑拔弩张
- 主角决定继续寻找父亲

**经过**：
1. **初到敦煌**：主角在敦煌遇到天山派弟子「雪女」
2. **天山派之邀**：雪女邀请主角前往天山派做客
3. **魔教来袭**：魔教袭击天山派，双方展开大战
4. **主角相助**：主角帮助天山派击退魔教
5. **天山派秘闻**：天山派掌门告知主角，父亲曾是天山派的客人
6. **魔教卧底**：主角发现天山派内部有魔教卧底
7. **深入魔教**：主角卧底魔教，发现魔教正在策划进攻中原
8. **父亲的下落**：主角在魔教总坛发现父亲被关押在「死亡谷」
9. **救出父亲**：主角与雪女一起前往死亡谷，救出父亲
10. **真相大白**：父亲告知主角，十年前的阴谋是由「暗影盟」策划的
11. **暗影盟**：暗影盟是一个由各大门派叛徒组成的神秘组织
12. **大逃亡**：主角等人被暗影盟追杀，一路逃回中原

**结果**：
- 救出父亲
- 得知暗影盟的存在
- 雪女加入队伍
- 开启第五章剧情

**关键NPC**：
1. **李慕白**（主角父亲）：凌云剑，十年前失踪，被暗影盟关押
2. **雪女**：天山派弟子，冷若冰霜，内心善良
3. **天山派掌门**：天山派掌门，武功高强
4. **魔教教主**：魔教教主，被暗影盟利用
5. **暗影使**：暗影盟高手，神出鬼没

**关键战斗**：
1. **魔教教徒×5**：天山派袭击者
2. **魔教护法**（BOSS）：魔教高手
3. **暗影使×2**：暗影盟高手
4. **死亡谷守卫×5**：死亡谷守卫
5. **暗影盟主**（BOSS）：暗影盟盟主（首次交手）

**分支选择**：
1. 是否加入天山派（影响势力选择）
2. 是否信任魔教教主（影响剧情）
3. 救父亲时是否牺牲雪女（影响结局）

**奖励**：
- 经验值：10000
- 金钱：2000两
- 物品：天山剑法（秘籍）、魔教心法（秘籍）、雪莲×3
- 声望：天山派+150、魔教+50（或-100，取决于选择）、敦煌+100
- 好感：雪女+60、李慕白+100

### 第五章：最终决战

**章节名称**：侠影江湖

**开启条件**：完成第四章，主角等级达到40级

**故事背景**：
主角带着父亲回到中原，发现暗影盟已经控制了江湖上的许多门派，正在策划一场惊天动地的阴谋——他们想要推翻朝廷，建立自己的武林帝国。主角必须联合江湖上的正义之士，一起对抗暗影盟。在这场决战中，主角将面临最终的选择，这些选择将决定江湖的命运，也将决定主角的结局。

**详细剧情流程**：

**起因**：
- 主角回到中原，发现暗影盟已经控制了许多门派
- 父亲告知主角，暗影盟的总坛在「隐龙山」
- 主角决定联合各大门派，一起对抗暗影盟

**经过**：
1. **联合各大门派**：主角游说各大门派，组成「正义联盟」
2. **暗影盟来袭**：暗影盟先发制人，袭击正义联盟
3. **首战告捷**：正义联盟击退暗影盟的进攻
4. **潜入隐龙山**：主角带领精锐，潜入隐龙山
5. **层层关卡**：主角闯过暗影盟的层层关卡
6. **最终对决**：主角在隐龙山顶峰与暗影盟主展开最终对决
7. **真相揭露**：暗影盟主竟然是主角的「叔叔」——李慕尘
8. **李慕尘的故事**：李慕尘讲述了自己的故事——当年他与李慕白同时爱上了一个女子，最终女子选择了李慕白，李慕尘因爱生恨，加入了暗影盟
9. **最终选择**：主角面临选择——是杀死李慕尘报仇，还是放过他
10. **结局分支**：根据主角的选择，进入不同的结局

**结果**：
- 暗影盟被消灭（或与主角和解）
- 江湖恢复和平（或陷入新的混乱）
- 主角成为一代大侠（或枭雄）
- 进入最终结局

**关键NPC**：
1. **李慕尘**（暗影盟主）：主角的叔叔，武功深不可测
2. **各大门派掌门**：正义联盟的领袖
3. **林月如、张天师、苏婉清、小昭、雪女**：主角的伙伴们
4. **暗影盟四大护法**：暗影盟的高手

**关键战斗**：
1. **暗影盟教徒×10**：正义联盟首战
2. **暗影盟四大护法×4**（BOSS）：隐龙山关卡
3. **李慕尘**（最终BOSS）：暗影盟主

**分支选择**：
1. 是否杀死李慕尘（影响结局）
2. 是否成为武林盟主（影响结局）
3. 选择哪位伴侣（影响结局）

**奖励**：
- 经验值：50000
- 金钱：10000两
- 物品：暗影剑法（秘籍）、凌云剑法（完整版）、九龙宝藏钥匙
- 声望：全江湖各门派+500
- 好感：所有伙伴+100

### 结局一：仁者无敌（完美结局）

**达成条件**：
- 道德值≥80
- 放过李慕尘
- 不成为武林盟主
- 林月如/张天师/苏婉清/小昭/雪女好感度≥80（至少一人）

**结局内容**：
主角放过了李慕尘，李慕尘深受感动，从此改邪归正，归隐山林。主角拒绝了武林盟主的位置，选择与心爱的人一起回到平安镇，过着平静的生活。但江湖上永远流传着主角的传说——「侠影江湖，仁者无敌」。多年后，主角的孩子长大成人，也踏上了江湖之路，新的故事又开始了……

**结局奖励**：
- 称号：「仁者大侠」
- 特殊物品：「仁者之心」（永久增加道德值）
- 解锁：隐藏剧情「新的征程」

### 结局二：武林霸主（枭雄结局）

**达成条件**：
- 道德值≤30
- 杀死李慕尘
- 成为武林盟主
- 声望≥5000

**结局内容**：
主角杀死了李慕尘，成为了武林盟主。但在权力的诱惑下，主角逐渐变得心狠手辣，铲除异己，建立了自己的武林帝国。江湖虽然统一，但却陷入了白色恐怖之中。主角站在隐龙山顶峰，俯视着自己的江山，心中却感到一丝寂寞——这真的是自己想要的吗？
