# 侠影江湖 — UI 设计文档

> 版本：1.0  
> 引擎：Godot 4.4  
> 风格：中国水墨画（水墨风）+ 仙侠游戏美术

---

## 1. UI 风格定义

### 1.1 设计理念

整体视觉以**中国传统水墨画**为核心语言，融合仙侠游戏的华丽感。界面追求"墨韵留白、金线勾勒"的审美——暗色底面如宣纸浸润，金色边线如工笔描金，红色点缀如朱砂落印。所有面板、按钮、图标均需保持水墨质感的一致性，避免出现扁平化或西式 UI 元素。

### 1.2 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景-深墨 | `#1a1a2e` | 最深层底色，用于全屏背景 |
| 主背景-暗蓝 | `#16213e` | 次深层底色，用于面板底色 |
| 主背景-中灰 | `#0f3460` | 中间层，用于次级面板 |
| 金色-主调 | `#c9a96e` | 边框、标题、重要文字 |
| 金色-高亮 | `#f0c040` | 按钮 hover、选中态、关键数值 |
| 红色-警示 | `#c0392b` | HP 条、伤害数字、警告提示 |
| 红色-朱砂 | `#e74c3c` | 朱砂印章、重要标记 |
| 翡翠绿 | `#2ecc71` | MP 条、治疗数字、正面状态 |
| 青蓝色 | `#3498db` | 内力条、水系技能、信息提示 |
| 紫色-神秘 | `#9b59b6` | 那迦派主题色、稀有物品 |
| 白色-文字 | `#f5f0e8` | 正文文字，带微暖色调 |
| 灰色-辅助文字 | `#a09888` | 次要说明文字 |

**渐变规则：**
- 面板背景：从 `#1a1a2e` 到 `#16213e` 的垂直渐变，透明度 85%-92%
- HP 条：从 `#c0392b` 到 `#e74c3c` 的水平渐变
- MP 条：从 `#2980b9` 到 `#2ecc71` 的水平渐变
- 金色边框：从 `#c9a96e` 到 `#f0c040` 的 45° 渐变

### 1.3 字体规范

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| 标题/Logo | 方正清刻本悦宋 | 48-72px | Bold |
| 面板标题 | 思源宋体 (Noto Serif SC) | 24-32px | Bold |
| 正文 | 思源宋体 (Noto Serif SC) | 16-18px | Regular |
| 数值/属性 | 思源黑体 (Noto Sans SC) | 14-16px | Medium |
| 按钮文字 | 思源宋体 (Noto Serif SC) | 18-20px | Bold |
| 提示/说明 | 思源黑体 (Noto Sans SC) | 12-14px | Regular |

**字体加载策略：**
- 标题字体仅加载 Bold 字重，正文字体加载 Regular + Bold
- 使用 Godot 的 `DynamicFont` 资源，支持中文子集化以减小包体
- 备选字体：系统宋体 / 微软雅黑（低配设备降级方案）

### 1.4 面板样式

所有面板遵循统一的视觉规范：

```
┌──────────────────────────────────┐  ← 金色描边 (#c9a96e), 2px
│  ░░░ 半透明暗色底面 ░░░          │  ← 背景 #1a1a2e, Alpha 88%
│  ▓▓▓ 水墨纹理叠加 ▓▓▓           │  ← ink_texture.tres, Alpha 15%
│                                  │
│  ┌─ 标题栏 ─────────────────┐    │  ← 金色底线分隔
│  │  面板标题                 │    │
│  └──────────────────────────┘    │
│                                  │
│  内容区域                        │
│                                  │
└──────────────────────────────────┘  ← 四角可加金色角花装饰
```

**面板组件规范：**
- 圆角：8px
- 边框：2px 金色渐变描边
- 内边距：16px
- 面板间距：12px
- 阴影：偏移 (4, 4)，模糊 12，颜色 `#00000080`
- 水墨纹理：使用 `ink_wash_texture.png` 叠加，透明度 10%-20%

### 1.5 按钮样式

| 状态 | 背景 | 边框 | 文字色 |
|------|------|------|--------|
| Normal | `#16213e` Alpha 80% | `#c9a96e` 1px | `#f5f0e8` |
| Hover | `#1a2a4e` Alpha 90% | `#f0c040` 2px | `#f0c040` |
| Pressed | `#0f1e3e` Alpha 95% | `#c9a96e` 1px | `#c9a96e` |
| Disabled | `#0a0a1e` Alpha 60% | `#555555` 1px | `#666666` |

**按钮装饰：**
- 左右两侧添加毛笔笔触装饰（brush_stroke_left.png / brush_stroke_right.png）
- 选中态按钮底部添加朱砂印章标记

### 3.4 动画与过渡

| 场景 | 动画类型 | 时长 | 缓动 |
|------|----------|------|------|
| 面板打开 | 从底部滑入 + 淡入 | 0.3s | EaseOut |
| 面板关闭 | 向底部滑出 + 淡出 | 0.2s | EaseIn |
| 场景切换 | 水墨晕染遮罩过渡 | 0.8s | EaseInOut |
| 战斗进入 | 墨迹飞溅 + 画面撕裂 | 0.5s | EaseOut |
| 对话出现 | 文字逐字显示 | 30字/s | - |
| 伤害数字 | 上浮 + 缩放 + 淡出 | 1.2s | EaseOut |
| HP/MP 变化 | 条形平滑过渡 | 0.3s | Linear |
| 按钮按下 | 缩放 0.95 + 颜色变化 | 0.1s | EaseOut |
| 小地图脉冲 | 玩家标记呼吸动画 | 2s 循环 | EaseInOut |

### 3.5 响应式适配

| 分辨率 | 策略 |
|--------|------|
| 1920×1080 | 基准分辨率，所有 UI 按此设计 |
| 1440×900 | 缩放 0.85，面板等比缩小 |
| 2560×1440 | 缩放 1.2，面板等比放大 |
| 3840×2160 | 缩放 1.5，确保文字清晰 |
| 1280×720 | 缩放 0.75，简化部分装饰元素 |

**Godot 实现方式：**
- 使用 `CanvasItem` 的 `_gui_input` 处理交互
- 使用 `Theme` 资源统一管理字体、颜色、样式
- 使用 `Control` 节点的 `anchor` 和 `margin` 实现自适应布局
- 使用 `ShaderMaterial` 实现水墨晕染过渡效果
