> 版本：2.0  
> 引擎：Godot 4.4

## 一、UI风格定义

### 1.1 视觉风格理念

整体视觉以**中国传统水墨画**为核心语言，融合仙侠游戏的华丽感。界面追求"墨韵留白、金线勾勒"的审美——暗色底面如宣纸浸润，金色边线如工笔描金，红色点缀如朱砂落印。所有面板、按钮、图标均需保持水墨质感的一致性，避免出现扁平化或西式 UI 元素。

**核心要素：**
- **墨韵层次**：浓淡干湿五色墨，营造立体层次感
- **留白意境**：适当留白，突出主体内容
- **金线勾边**：金色边框和文字作为点缀，提升华贵感
- **朱砂印信**：红色用于强调、状态和重要提示
- **动态墨染**：使用水墨晕染、墨迹飞溅等动态效果

- **MP条**：`#2980b9` → `#2ecc71` 水平渐变
- **金色边框**：`#c9a96e` → `#f0c040` 45°渐变
- **能量条**：根据属性使用对应渐变

- 备选字体：系统宋体 / 微软雅黑（低配设备降级方案）

### 界面堆栈管理

**栈结构**：
```
栈底
  ├─ 主菜单
  ├─ 角色创建
  ├─ 游戏主场景 (HUD常驻)
  │   ├─ 背包
  │   ├─ 技能
  │   ├─ 任务
  │   ├─ 对话
  │   ├─ 战斗 (顶置)
  │   └─ ...
  └─ ...
栈顶
```

**堆栈操作**：
- **Push(界面)**：将新界面压入栈顶，显示新界面，暂停下层界面
- **Pop()**：弹出栈顶界面，恢复下层界面
- **Replace(界面)**：替换栈顶界面
- **ClearTo(界面)**：清空到指定界面
- **GetTop()**：获取当前栈顶界面

### 界面过渡动画

**打开动画**：
```
淡入 + 从底部滑入
时长：0.3秒
缓动：EaseOutQuad
```

**关闭动画**：
```
淡出 + 向底部滑出
时长：0.2秒
缓动：EaseInQuad
```

**场景切换**：
```
水墨晕染覆盖 → 加载 → 水墨晕染消失
时长：0.8秒
缓动：EaseInOutQuad
```

**战斗进入**：
```
墨迹飞溅 → 画面撕裂 → 战斗界面出现
时长：0.5秒
缓动：EaseOutBack
```

**面板交换**：
```
当前面板向左滑出 + 新面板从右滑入
时长：0.4秒
缓动：EaseInOutQuad
```
