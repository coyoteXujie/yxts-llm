# 一、物品体系总纲

## 1.1 物品分类体系

本游戏物品分为八大类，与代码中 `ItemType` 枚举严格对应：

| 分类 | 代码枚举值 | 说明 | 示例 |
|------|-----------|------|------|
| 消耗品 | `CONSUMABLE` | 使用后消失，恢复生命/内力/提供临时效果 | 金创药、小还丹、大力丸 |
| 武器 | `WEAPON` | 提供攻击力加成，装备在武器槽 | 铁剑、八卦刀、木剑 |
| 防具 | `ARMOR` | 提供防御力加成，分为头/身/手/足/腰/饰品 | 布衣、皮甲、金丝软甲 |
| 材料 | `MATERIAL` | 用于锻造、炼丹、缝纫等制作系统 | 铁矿石、灵芝、丝绸 |
| 书籍 | `BOOK` | 包括武学秘籍和杂书，可学习新技能 | 太极拳谱、论语 |
| 任务物品 | `QUEST` | 完成任务所需，不可交易 | 信物、密函、钥匙 |

> **代码对照**: `shared_types.py` 中 `ItemType` 枚举定义了以上六种类型。`equipment_system.py` 中 `Item` 数据类额外包含 `slot`、`attack`、`defense`、`speed`、`hp_bonus`、`mp_bonus`、`effects`、`enhance_level`、`max_enhance` 等装备属性字段。

## 1.2 物品稀有度等级

与代码中 `ItemRarity` 枚举对应，共5级：

| 等级 | 代码枚举 | 名称 | 颜色标识 | 描述风格 | 掉落率 |
|------|---------|------|---------|---------|--------|
| 1 | `COMMON` | 普通 | 白色 | 平淡朴实，"一把普通的铁剑" | 60% |
| 2 | `UNCOMMON` | 优秀 | 绿色 | 略有特色，"精钢打造的长剑" | 25% |
| 3 | `RARE` | 稀有 | 蓝色 | 难得一见，"暗藏八卦纹路" | 10% |
| 4 | `EPIC` | 史诗 | 紫色 | 极其珍贵，"传说中XXX所铸" | 4% |
| 5 | `LEGENDARY` | 传说 | 金色 | 举世无双，"武林至尊，号令天下" | 1% |

### 稀有度视觉差异

- **普通(白)**: 物品名称白色显示，描述简洁直白，无特殊光效
- **优秀(绿)**: 物品名称绿色显示，描述中带有品质修饰词，轻微绿色光晕
- **稀有(蓝)**: 物品名称蓝色显示，描述中暗示来历或特殊材质，蓝色微光
- **史诗(紫)**: 物品名称紫色显示，描述中涉及传说人物或重大事件，紫色流光
- **传说(金)**: 物品名称金色显示，描述充满传奇色彩，金色光芒环绕，获取时有特殊公告

## 1.4 物品鉴定系统

部分物品（尤其是从怪物掉落和宝箱中获得的装备）可能处于"未鉴定"状态：

| 鉴定状态 | 说明 |
|---------|------|
| 未鉴定 | 物品名称显示为"???"，无法装备，属性未知 |
| 已鉴定 | 显示完整名称和属性，可正常使用 |

### 鉴定方式

| 方式 | 费用 | 成功率 | 说明 |
|------|------|--------|------|
| 自行鉴定 | 消耗鉴定符x1 | 50% | 鉴定符可在杂货铺购买 |
| 找鉴定师 | 100-1000文（视稀有度） | 100% | 城镇中的鉴定师NPC |
| 使用鉴定术 | 消耗内力50点 | 70%+悟性加成 | 需要学习鉴定技能 |

### 鉴定结果

- 鉴定成功：物品变为已鉴定状态，显示完整属性
- 鉴定失败：物品仍为未鉴定状态，鉴定符/金钱不退还
- 特殊鉴定：传说物品鉴定时有5%概率触发"天降异象"，物品获得额外属性加成

## 1.5 物品耐久度系统

### 耐久度机制

所有装备类物品（武器、防具）拥有耐久度属性：

| 属性 | 说明 |
|------|------|
| 最大耐久 | 装备的最大耐久值，与品质相关 |
| 当前耐久 | 装备的当前耐久值，使用中逐渐降低 |
| 耐久度百分比 | 当前耐久/最大耐久 |

### 耐久度消耗

| 行为 | 耐久消耗 |
|------|---------|
| 普通攻击 | -1点 |
| 使用武学招式 | -2点 |
| 被攻击（防具） | -1点 |
| 暴击攻击 | -3点 |
| 被暴击（防具） | -2点 |

### 耐久度效果

| 耐久度范围 | 效果 |
|-----------|------|
| 100%-60% | 正常状态，属性100%发挥 |
| 59%-30% | 轻微磨损，属性降低10% |
| 29%-1% | 严重磨损，属性降低30% |
| 0% | 装备损坏，无法使用，需要修理 |

### 修理系统

| 修理方式 | 费用 | 效果 |
|---------|------|------|
| 铁匠修理 | 金钱 = 最大耐久 * 稀有度系数 | 恢复至最大耐久 |
| 自行修理 | 消耗铁矿石x2 + 皮革x1 | 恢复50%耐久 |
| 修理符 | 消耗修理符x1 | 恢复30%耐久 |

# 五、套装系统

## 5.1 套装机制说明

当玩家同时装备同一套装的多件装备时，可触发套装效果。套装效果分为多个等级，装备件数越多效果越强。

### 套装效果叠加规则

```
套装效果 = 基础效果(2件) + 进阶效果(4件) + 终极效果(6件)
```

- 2件：触发基础效果
- 4件：触发基础+进阶效果
- 6件（全套）：触发基础+进阶+终极效果（通常包含专属技能）

# 七、锻造系统详解

## 7.1 锻造技能等级

代码中 `CraftingSkill` 类定义了锻造技能的等级系统：

| 等级 | 称号 | 所需经验 | 可制作物品 | 备注 |
|------|------|---------|-----------|------|
| 1-2 | 学徒 | level * 100 | 基础装备(铁剑、皮甲) | 成功率较高(80-90%) |
| 3-4 | 匠人 | level * 100 | 中级装备(钢剑、锁子甲) | 成功率中等(60-80%) |
| 5-7 | 大师 | level * 100 | 高级装备(精钢剑、精钢甲) | 成功率较低(40-60%) |
| 8-10 | 宗师 | level * 100 | 传说装备(玄铁重剑) | 成功率极低(20-40%) |

### 经验值获取

```
成功打造: +20 + recipe.skill_level * 10 经验
打造失败: +10 经验
升级所需: skill.level * 100 经验
```

> **代码对照**: `CraftingSystem` 中定义了四种打造技能：锻造(blacksmith)、炼丹(alchemy)、制衣(tailor)、烹饪(cooking)。

## 7.2 锻造配方树

### 基础材料 → 中级材料 → 高级成品

```
铁矿石(x5) + 皮革(x1) ──锻造──→ 铁剑(attack=15)
                                       ↓
铁矿石(x2) + 钢材(x3) + 皮革(x2) ──锻造──→ 钢剑(attack=25)
                                                ↓
精钢矿石(x5) + 钢材(x3) ──锻造──→ 精钢剑(attack=35)
                                       ↓
寒铁(x5) + 钢材(x3) ──锻造──→ 寒铁剑(attack=50)
                                    ↓
玄铁(x3) + 寒铁(x5) + 天蚕丝(x2) ──锻造──→ 玄铁重剑(attack=70)
```

### 炼丹配方树

```
普通草药(x2) ──炼丹──→ 金创药x3(heal_50)
普通草药(x3) + 珍稀草药(x1) ──炼丹──→ 小还丹x2(heal_150)
珍稀草药(x5) + 灵芝(x1) ──炼丹──→ 大还丹x1(heal_full)
```

> **代码对照**: 以上配方与代码中 `RECIPES` 字典完全对应。

## 7.3 锻造成功率与影响因素

### 基础成功率

```
最终成功率 = min(1.0, recipe.base_success_rate + (skill.level - recipe.skill_level) * 0.05)
```

### 影响因素

| 因素 | 影响 | 说明 |
|------|------|------|
| 技能等级 | +5%/超出等级 | 技能等级每超过配方要求1级，成功率+5% |
| 材料品质 | +5%~+20% | 使用高品质材料可提升成功率 |
| 工具品质 | +5%~+15% | 使用高级锻造工具可提升成功率 |
| 天气 | -10%~+10% | 晴天+5%，雨天-5%，雷雨天+10%(雷电锻造) |
| 运气 | 随机波动 | 最终结果有随机波动范围 |

### 成功率示例

以锻造钢剑为例（`base_success_rate=0.7`，`skill_level=3`）：

| 锻造等级 | 计算 | 成功率 |
|---------|------|--------|
| 3级(刚好) | 0.7 + (3-3)*0.05 | 70% |
| 4级 | 0.7 + (4-3)*0.05 | 75% |
| 5级 | 0.7 + (5-3)*0.05 | 80% |
| 8级 | 0.7 + (8-3)*0.05 = 0.95 → min(1.0, 0.95) | 95% |

## 7.4 锻造品质系统

成功锻造后，物品品质在 `quality_range` 范围内随机决定：

```
品质 = random.randint(quality_min, quality_max)
属性加成 = 基础属性 * (1 + (品质 - 1) * 0.2)
价值 = 基础价值 * 品质
```

| 品质 | 属性加成 | 价值倍率 | 颜色 |
|------|---------|---------|------|
| 1(普通) | 100% | 1x | 白色 |
| 2(优秀) | 120% | 2x | 绿色 |
| 3(稀有) | 140% | 3x | 蓝色 |
| 4(史诗) | 160% | 4x | 紫色 |
| 5(传说) | 180% | 5x | 金色 |
