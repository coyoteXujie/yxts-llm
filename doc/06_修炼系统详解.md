# 修炼系统详解

> 本文档基于 `src/core/systems/cultivation_system.py` 源代码，深度解析"白金英雄坛说"的武学修炼体系。本系统融合传统武侠文化精髓与现代游戏设计理念，为玩家提供沉浸式的武学修行体验。

---

## 一、修炼系统概述

### 1.1 武侠修炼的哲学基础

在中国传统武侠文化中，武学修炼不仅是技艺的精进，更是心性的磨练与境界的提升。"白金英雄坛说"的修炼系统深刻体现了这一哲学内核：

**内外兼修之道**

武学修炼分为"内功"与"外功"两大体系。内功主修内力，是武学根基；外功主修招式，是实战手段。正如武学经典所言："内练一口气，外练筋骨皮。"只有内外兼修，方能成为真正的武林高手。

**循序渐进之理**

修炼系统遵循"由浅入深、由易到难"的原则。玩家需要从基础功法开始，逐步打通经脉、提升境界，最终达到武学巅峰。这一过程模拟了真实武学修行的艰辛与收获。

**天人合一之境**

最高境界的武学修炼追求"天人合一"——人与自然的和谐统一。当玩家将内功修炼至化境，轻功达到极致，经脉全部贯通时，便能领悟这一至高境界，获得超凡脱俗的力量。

### 1.2 系统架构总览

修炼系统由五大核心子系统构成，形成完整的武学成长体系：

| 子系统 | 核心功能 | 成长维度 | 哲学内涵 |
|--------|----------|----------|----------|
| **内功修炼** | 提升内力上限、恢复速度、属性加成 | 内力深度 | 固本培元，以内驭外 |
| **外功招式** | 各门派武学、招式熟练度 | 技艺广度 | 招式变化，以巧破力 |
| **轻功身法** | 移动速度、闪避、特殊移动能力 | 身法灵动 | 动如脱兔，静若处子 |
| **绝招系统** | 强力技能、冷却机制 | 爆发力 | 蓄势待发，一击必杀 |
| **经脉系统** | 经脉打通、永久属性加成 | 生命层次 | 打通玄关，脱胎换骨 |

### 1.3 修炼境界体系

武学修炼分为五个大境界，每个境界代表着不同的修为层次：

| 境界 | 名称 | 特征描述 | 解锁能力 |
|------|------|----------|----------|
| 第一境 | **初窥门径** | 刚入门派，学习基础功法 | 基础内功、轻功 |
| 第二境 | **小有所成** | 功法初成，可独当一面 | 进阶内功、外功招式 |
| 第三境 | **炉火纯青** | 技艺纯熟，门派中坚 | 绝学招式、经脉修炼 |
| 第四境 | **登峰造极** | 一派宗师，开宗立派 | 自创武学、传承弟子 |
| 第五境 | **天人合一** | 超凡入圣，武道巅峰 | 破碎虚空、武道真意 |

## 二、武学类型详解

### 2.1 四大武学类型

游戏将所有武学技能分为四大类型，每种类型代表不同的武学修炼方向：

| 类型 | 枚举值 | 核心作用 | 修炼重点 | 代表功法 |
|------|--------|----------|----------|----------|
| **内功** | `INTERNAL` | 提升内力上限、恢复速度和基础属性 | 打坐冥想、吐纳调息 | 北冥神功、太极功 |
| **外功** | `EXTERNAL` | 各门派攻击招式，用于战斗输出 | 招式练习、实战磨练 | 八卦掌、花间剑法 |
| **轻功** | `LIGHTNESS` | 提升移动速度和闪避率，解锁特殊移动 | 身法练习、提气纵跃 | 凌波微步、踏雪无痕 |
| **绝招** | `ULTIMATE` | 强力终极技能，具有冷却机制 | 内力凝聚、意境领悟 | 混元一气、红莲业火 |

### 2.2 内功修炼之道

内功是武学根基，决定了武者的内力深厚程度。修炼内功时，玩家需要：

**打坐调息**：寻找安静场所，盘膝而坐，调整呼吸，引导内力在经脉中运行。

**吐纳炼气**：通过特定的呼吸法门，吸收天地灵气，转化为自身内力。

**心法领悟**：研读内功心法，理解其中的武学至理，提升修炼效率。

### 2.3 外功修炼之道

外功是实战手段，决定了武者的攻击能力。修炼外功时，玩家需要：

**招式练习**：反复练习招式动作，直至熟练自如，形成肌肉记忆。

**对练切磋**：与其他武者切磋，在实战中检验招式，发现不足。

**意境领悟**：理解招式背后的武学意境，达到"意到招随"的境界。

### 2.4 轻功修炼之道

轻功是身法技艺，决定了武者的移动能力和闪避能力。修炼轻功时，玩家需要：

**负重练习**：身负重物进行跳跃、奔跑，锻炼腿部力量。

**平衡训练**：在狭窄、不稳定的表面行走，提升身体平衡能力。

**提气轻身**：学习提气法门，使身体轻盈如燕，踏雪无痕。

### 2.5 绝招修炼之道

绝招是压箱底的绝技，威力巨大但消耗也大。修炼绝招时，玩家需要：

**内力储备**：绝招需要消耗大量内力，必须确保内力充足。

**时机把握**：绝招有冷却时间，必须在关键时刻使用。

**意境凝聚**：绝招往往蕴含特殊的武学意境，需要用心领悟。

## 三、八大内功详解

内功由 `InternalArt` 数据类定义，每个内功包含内力属性、属性加成和特殊效果。以下是对八大内功的深度解析：

### 3.1 内功基础属性说明

| 属性 | 说明 | 计算公式 |
|------|------|----------|
| `mp_base` | 基础内力上限 | 内功初始提供的内力值 |
| `mp_per_level` | 每级增加的内力上限 | 默认20，北冥神功为30 |
| `mp_regen` | 每级内力恢复速度 | 每回合恢复的内力值 |
| `hp_bonus` | 每级生命值加成 | 提升最大生命值 |
| `attack_bonus` | 每级攻击力加成 | 提升攻击力 |
| `defense_bonus` | 每级防御力加成 | 提升防御力 |
| `speed_bonus` | 每级速度加成 | 提升出手速度 |
| `max_level` | 最高等级 | 默认为9级 |

### 内力上限计算公式

```
max_mp = mp_base + mp_per_level * level
```

### 3.2 混元功（八卦门）

**功法概述**

混元功是八卦门的基础内功心法，讲究"中正平和、混元一体"。此功法源自道家修炼法门，强调阴阳调和、动静结合。修炼者通过吐纳调息，将天地灵气转化为自身内力，逐步壮大丹田之气。

**完整修炼心法描述**

修炼混元功时，修炼者需于每日清晨面向东方，盘膝而坐，双手结混元印置于丹田。初时只觉气息紊乱，难以入静，但随着修炼深入，渐渐能感受到一股温热之气自丹田升起，沿任督二脉缓缓运行。每运行一周天，那股气息便壮大一分，如涓涓细流汇入江河。

当修炼至第三层时，修炼者会感受到体内气息已形成完整循环，举手投足间自有内力相随。到了第六层，丹田中仿佛有一团混元之气在缓缓旋转，生生不息。而达到第九层圆满境界时，修炼者周身气息浑然一体，内外交融，已达"混元一体"之境。此时修炼者面色红润，气息绵长，即便数日不眠不休，精神依旧饱满。

**与其他内功的兼容性/冲突**

混元功作为最中正平和的内功，与几乎所有内功都兼容。它是修炼其他进阶内功的最佳基础，尤其适合作为初学者的第一门内功。修炼混元功后，转修其他内功时不会有任何阻碍。

**修炼时的视觉效果描述**

修炼混元功时，修炼者周身会泛起淡淡的白色光晕，如晨雾般朦胧。随着等级提升，光晕会逐渐变得凝实。达到高阶时，修炼者头顶会有白气升腾，如炊烟袅袅，这是体内浊气排出的表现。

### 3.3 八卦内功（八卦门）

**功法概述**

八卦内功是八卦门的进阶内功，在混元功的基础上融入了八卦变化之理。此功法讲究"变化无穷、生生不息"，内力运行时遵循八卦方位，形成独特的八卦气场。

**完整修炼心法描述**

修炼八卦内功需在特定的八卦图案上打坐，按照"乾、坤、震、巽、坎、离、艮、兑"八方的顺序引导内力运行。初修时，修炼者需在地上绘制八卦图，按图修炼；待功法有成，心中自有八卦，随时随地皆可修炼。

修炼至第三层时，修炼者能感受到体内内力按照八卦方位流转，形成八个气旋。到了第六层，八个气旋相互连接，形成完整的八卦气场，攻守兼备。达到第九层圆满时，八卦气场已臻化境，修炼者周身仿佛笼罩着一层无形的气罩，可自动抵御外敌攻击。

**修炼时的视觉效果描述**

修炼八卦内功时，修炼者周身会出现八个淡淡的光点，按照八卦方位排列。随着修炼深入，光点之间会有光线连接，形成完整的八卦图案。达到高阶时，八卦图案会缓缓旋转，散发出金色光芒。

### 3.4 花心诀（百花谷）

**功法概述**

花心诀是百花谷的独门内功，由花间派祖师所创。此功法讲究"如花绽放、生生不息"，以内力模拟花开花落的生命循环，具有独特的恢复和灵动特性。

**完整修炼心法描述**

修炼花心诀需在花丛之中，感受花朵从含苞到绽放再到凋零的全过程。修炼者需将自身内力想象成一朵花，按照"含苞、初绽、盛放、凋零、再生"五个阶段循环运行。

初修时，修炼者会感到内力如种子般在丹田中孕育，缓慢而艰难。到了第三层，内力如花朵初绽，开始展现出灵动的特性。第六层时，内力如盛放的鲜花，充满生机与活力，恢复能力大幅提升。达到第九层圆满，内力已达到"花落花开、生生不息"的境界，即便身受重伤，也能快速恢复。

### 3.5 红莲诀（红莲教）

**功法概述**

红莲诀是红莲教的镇教内功，炽热如火，霸道无比。此功法讲究"炽热红莲、焚尽万物"，以内力模拟红莲业火，具有极强的攻击性和破坏力。

**完整修炼心法描述**

修炼红莲诀需在炎热之地，感受火焰的炽热与毁灭之力。修炼者需将自身内力想象成一朵红莲，在丹田中熊熊燃烧。初修时，修炼者会感到丹田如被火焚，痛苦难忍，必须以大毅力坚持下去。

到了第三层，修炼者已能适应那股炽热，内力中开始带有火属性。第六层时，红莲之火已壮大到可以外放，举手投足间带有灼热之气。达到第九层圆满，修炼者周身仿佛笼罩着一层红莲业火，所过之处草木焦枯，威力惊人。

### 3.6 那迦诀（那迦派）

**功法概述**

那迦诀是那迦派的独门内功，讲究"隐忍如影、动若脱兔"。此功法源自西域武学，以内力强化身法，使修炼者行动如毒蛇般迅捷诡秘。

**完整修炼心法描述**

修炼那迦诀需在阴暗之处，模仿毒蛇的蛰伏与突袭。修炼者需将自身内力想象成一条那迦（毒蛇），平时蛰伏于丹田，一旦发动则迅若闪电。

## 四、五大轻功详解

轻功由 `LightnessArt` 数据类定义，提供移动速度、闪避率和特殊移动能力。

### 4.1 轻功属性说明

| 属性 | 说明 | 计算公式 |
|------|------|----------|
| `speed_base` | 基础移动速度倍率 | 初始移动速度 |
| `speed_per_level` | 每级增加的速度 | 默认0.1 |
| `dodge_base` | 基础闪避率 | 初始闪避率 |
| `dodge_per_level` | 每级增加的闪避率 | 默认2 |
| `can_water_walk` | 是否可水上行走 | 特殊能力 |
| `can_wall_run` | 是否可飞檐走壁 | 特殊能力 |
| `can_double_jump` | 是否可二段跳 | 特殊能力 |

### 速度计算公式

```
speed = speed_base + speed_per_level * level
```

### 闪避计算公式

```
dodge = dodge_base + dodge_per_level * level
```

### 5.6 突破描述（每条经脉200字+）

#### 任脉突破

当你开始冲击任脉时，首先感受到的是丹田中一股温热之气缓缓升起。这股气息如涓涓细流，沿着腹部正中缓缓上行。突破的过程并不痛苦，反而有一种温暖的舒适感。当气息终于冲破任脉的最后一道关卡时，你会感到全身一轻，仿佛卸下了千斤重担。内力在任脉中流转自如，如江河入海，滔滔不绝。此时你的内力上限大幅提升，内力恢复速度也明显加快。这是修炼之路的第一步，也是最基础的一步。任脉打通后，你才真正踏入了武学修炼的大门。

#### 督脉突破

督脉的突破比任脉要艰难得多。当你引导内力进入督脉时，会感到一股灼热之气从会阴处升起，沿着脊柱急速上行。这股气息如火焰般炽热，所过之处仿佛有烈火在燃烧。突破的过程痛苦万分，你需要咬紧牙关，以大毅力坚持下去。当气息终于冲破督脉的顶端——百会穴时，你会感到头顶一凉，一股清凉之气从天而降，与上升的阳气交汇。这一刻，阴阳交汇，天人合一。你的攻击力和防御力都得到了显著提升，整个人仿佛脱胎换骨。

#### 冲脉突破

冲脉的突破是一场气血的洗礼。当你开始冲击冲脉时，会感到全身气血翻涌，如江河决堤。冲脉被称为"十二经之海"，是气血汇聚之地。突破时，你会感到气血在全身快速流转，冲刷着每一寸经脉。这个过程既痛苦又舒畅——痛苦的是气血冲击带来的胀痛，舒畅的是堵塞的经脉被一一打通。当冲脉终于贯通时，你会感到全身充满了力量，生命力大幅提升。你的生命值和生命恢复速度都得到了显著增强，仿佛年轻了十岁。

#### 带脉突破

带脉的突破是一场关于约束与释放的修行。带脉如腰带般环绕腰部，约束着纵行诸经。突破带脉时，你会感到腰部有一股强大的束缚力，仿佛被一条无形的腰带紧紧勒住。你需要用内力不断冲击这条"腰带"，直到它最终断裂。突破的瞬间，你会感到全身一松，仿佛脱去了束缚。但紧接着，你会发现自己的行动变得更加灵活，速度和闪避能力都得到了提升。带脉的打通让你明白了一个道理：真正的自由来自于适度的约束。

#### 阳跷脉突破

阳跷脉的突破是一场关于速度与爆发的觉醒。当你开始冲击阳跷脉时，会感到双腿中有一股炽热之气在涌动。这股气息如火焰般炽热，让你的双腿充满了力量。突破的过程需要你不断奔跑、跳跃，将那股炽热之气引导至全身。当阳跷脉终于贯通时，你会感到双腿轻盈如羽，仿佛随时可以飞起来。你的暴击率和暴击伤害都得到了显著提升，每一次攻击都充满了爆发力。阳跷脉的打通让你在战斗中更加致命。

#### 阴跷脉突破

阴跷脉的突破是一场关于反应与防御的修行。与阳跷脉的炽热不同，阴跷脉的气息阴冷而沉稳。当你开始冲击阴跷脉时，会感到双腿中有一股清凉之气在流动。这股气息如流水般柔和，让你的反应变得更加敏捷。突破的过程需要你保持冷静，在静中感悟。当阴跷脉终于贯通时，你会感到自己的反应速度大幅提升，能够轻松格挡敌人的攻击并进行反击。阴跷脉的打通让你在战斗中更加从容。

#### 阳维脉突破

阳维脉的突破是一场关于力量与伤害的升华。阳维脉维系着诸阳经，是阳气的汇聚之地。当你开始冲击阳维脉时，会感到全身阳气沸腾，如烈火燃烧。这股阳气不断冲击着你的经脉，带来剧烈的疼痛。但你需要坚持下去，因为只有经历过烈火的洗礼，才能获得真正的力量。当阳维脉终于贯通时，你会感到自己的内力变得无比凝实，每一次攻击都蕴含着强大的破坏力。你的内功伤害和技能伤害都得到了显著提升。

#### 阴维脉突破

阴维脉的突破是一场关于防御与抗性的修行。阴维脉维系着诸阴经，是阴气的汇聚之地。当你开始冲击阴维脉时，会感到全身被一股清凉之气包围。这股气息如护盾般保护着你，让你对外界的伤害产生了抗性。突破的过程需要你保持内心的平静，在静中感悟防御的真谛。当阴维脉终于贯通时，你会感到自己的防御力大幅提升，对各种负面状态的抗性也显著增强。阴维脉的打通让你在面对强敌时更加从容不迫。

## 八、技能熟练度系统

外功招式由 `MartialSkill` 数据类定义，具有等级和熟练度机制。

### 8.1 技能等级

| 属性 | 说明 |
|------|------|
| 最低等级 | 1 |
| 最高等级 | 10 |
| 精通状态 | `level >= max_level` 时为精通（`is_mastered`） |

### 8.2 技能属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `id` | `str` | 技能唯一标识 |
| `name` | `str` | 技能名称 |

### 8.3 伤害计算公式

```
技能伤害 = damage_base + damage_scale * level * 10
```

### 8.4 技能升级消耗

```
升级消耗 = 当前等级 * 100 修炼经验
```

| 当前等级 | 升级到下一级消耗 |
|----------|------------------|
| 1 -> 2 | 100 |
| 2 -> 3 | 200 |
| 3 -> 4 | 300 |
| 4 -> 5 | 400 |
| 5 -> 6 | 500 |
| 6 -> 7 | 600 |
| 7 -> 8 | 700 |
| 8 -> 9 | 800 |
| 9 -> 10 | 900 |

**从1级升到10级总消耗**：100+200+300+400+500+600+700+800+900 = **4500 修炼经验**

### 8.5 技能升级条件

技能升级需同时满足以下条件：

1. 技能等级未达上限（`level < max_level`）
2. 修炼经验充足（`cultivation_exp >= cost`）
3. 满足属性需求（`requirements` 字典中的所有属性要求）

## 九、修炼经验获取途径

### 9.1 修炼等级系统

| 属性 | 说明 |
|------|------|
| `cultivation_exp` | 当前修炼经验 |
| `cultivation_level` | 当前修炼等级（初始为1） |

### 9.2 修炼等级升级公式

```
升级所需经验 = cultivation_level * 1000
```

| 修炼等级 | 升级所需经验 | 累计经验 |
|----------|-------------|----------|
| 1 -> 2 | 1000 | 1000 |
| 2 -> 3 | 2000 | 3000 |
| 3 -> 4 | 3000 | 6000 |
| 4 -> 5 | 4000 | 10000 |
| 5 -> 6 | 5000 | 15000 |
| ... | ... | ... |

### 9.3 经验获取方式

修炼经验通过 `add_cultivation_exp(exp)` 方法增加，支持超额经验自动连续升级：

```python
def add_cultivation_exp(self, exp: int) -> None:
    self.cultivation_exp += exp
    exp_needed = self.cultivation_level * 1000
    while self.cultivation_exp >= exp_needed:
        self.cultivation_exp -= exp_needed
        self.cultivation_level += 1
        exp_needed = self.cultivation_level * 1000
```

### 9.4 修炼经验消耗总览

| 消耗项目 | 单次消耗 | 说明 |
|----------|----------|------|
| 技能升级 | `level * 100` | 外功招式升级 |
| 内功升级 | `level * 200` | 内功心法升级 |
| 经脉打通 | `500 * (已打通数 + 1)` | 首次打通经脉 |
| 经脉修炼 | 浮点经验值 | 提升经脉打通层数 |

## 十一、修炼与装备的联动

### 11.1 修炼系统接口

修炼系统通过 `link_player(player)` 方法关联玩家对象，使修炼系统能够访问角色属性数据：

```python
def link_player(self, player) -> None:
    self._player_ref = player
```

### 11.2 装备联动方式

修炼系统的 `get_total_stats()` 和 `get_bonuses()` 方法返回纯属性加成字典，这些加成会与装备提供的属性叠加，共同构成角色的最终战斗属性。

```
角色最终属性 = 基础属性 + 修炼加成（get_total_stats）+ 装备加成
```

### 11.3 内功装备机制

内功系统支持"学习"和"装备"两个独立操作：

| 操作 | 方法 | 说明 |
|------|------|------|
| 学习内功 | `learn_internal_art(art_id)` | 将内功加入已知列表，首次学习自动装备 |
| 装备内功 | `equip_internal_art(art_id)` | 切换当前使用的内功，保留已有等级 |

**关键机制**：切换内功时，如果切换回之前已装备过的同一内功，会保留其等级进度。切换到新的内功时则从 1 级开始。

### 11.4 轻功装备机制

轻功与内功采用相同的"学习+装备"机制：

| 操作 | 方法 | 说明 |
|------|------|------|
| 学习轻功 | `learn_lightness_art(art_id)` | 将轻功加入已知列表，首次学习自动装备 |
| 装备轻功 | （通过重新学习实现） | 切换当前使用的轻功 |

### 11.5 可用技能获取

通过 `get_available_skills()` 方法获取所有等级大于 0 的已学武学招式：

```python
def get_available_skills(self) -> List[MartialSkill]:
    return [s for s in self.skills.values() if s.level > 0]
```

这些可用技能将直接出现在战斗界面的技能选择列表中。

## 十三、总结

"白金英雄坛说"的修炼系统是一个深度与广度兼备的武学成长体系。通过内功、外功、轻功、绝招四大武学类型，配合经脉系统的永久加成，玩家可以体验到从初学者到武林宗师的完整成长历程。

系统的核心设计理念包括：

1. **循序渐进**：从基础到进阶，从简单到复杂，让玩家逐步掌握武学精髓
2. **内外兼修**：内功与外功并重，轻功与绝招互补，形成完整的武学体系
3. **文化底蕴**：融合中医经络理论与传统武侠文化，增强游戏的文化深度
4. **策略选择**：八大内功、五大轻功各有特色，玩家需要根据战斗风格进行选择
5. **长期成长**：经脉系统提供长期的成长目标，让玩家有持续追求的动力

这一系统不仅为游戏提供了丰富的玩法内容，也为AI开发提供了清晰的接口和数据结构，是"白金英雄坛说"的核心系统之一。
